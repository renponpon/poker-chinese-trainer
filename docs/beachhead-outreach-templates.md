# ビーチヘッド向け依頼文テンプレ（LINE / X）

**用途**: マカオ・海外でポーカーする日本人（ビーチヘッド）および、その周辺にいる学習者への**紹介依頼**・**本人打診**。  
**チャネル**: 基本は **LINE**（長め・信頼補足OK）か **X DM**（短く・摩擦少なく）。

[`customer-discovery.md`](./customer-discovery.md) の **A-3（Layer A / B / C）** と対応させている。

| Layer | 誰に送るか | このファイルのセクション |
|-------|------------|--------------------------|
| **A** | 仲間・知人（紹介してもらう側） | 各カテゴリの「A: 紹介を頼む」 |
| **B** | 紹介者が転送する文（紹介先へ） | 「転送用（Layer B）」 |
| **C** | 紹介先から返信が来たあと | 「返信用（Layer C）」 |

---

## 送る前の共通チェック

- [ ] **売り込みでない**と一言入れる（インタビュー／雑談の目的が中国語学習の理解であること）
- [ ] **所要時間**（例: 30分）と **形式**（Zoom / 電話 / 対面のどれか）
- [ ] 本人に会うなら **録音の可否**は当日冒頭で改めて確認（事前DMでは「録音したい場合あり」と軽く触れる程度で可）
- [ ] 紹介を頼むときは **「1人だけで十分」** を明記（心理的ハードルを下げる）

**LINE と X の使い分け**

| | LINE | X |
|---|------|---|
| 長さ | 4〜8行でも可 | 短く。必要なら2通に分ける |
| トーン | 普段のトーンに合わせる | 絵文字は相手と自分の習慣に合わせる（無しでも可） |
| URL | 予約リンクをそのまま貼ってよい | 長いURLは短縮 or 「返信くれたら送る」でも可 |

`[名前]` `[予約URL]` `[自分の名前]` は送信前に差し替える。

## 1. 仲の良い友達

### A: 紹介を頼む（LINE）

```
[名前]、久しぶり！

中国語の勉強まわりで、ちゃんと話聞きたい人を探してて。
マカオとか海外でポーカーやってる人、またはその周りで中国語使ってる日本人、心当たりいる？

いるなら、その人にだけ転送してもらえると助かる。
売り込みじゃなくて30分くらい雑談ベースで聞きたいだけ。

転送用の文、下に貼るからコピペでOK。無理ならスルーで！
```

### A: 紹介を頼む（X DM）

```
[名前]、元気？

中国語の勉強のリサーチで、海外ポーカーまわりの日本人に話聞きたいんだけど、心当たりいたら1人だけ紹介してほしい。売り込みなし30分くらい。転送文あるから送るね
```

### 転送用（Layer B）— 友達が第三者に送る文

```
共通の友達の[自分の名前]から。

中国語の勉強（現場で言葉が出ないとか、アプリの使い分け）について、経験ある人の話を集めてるらしい。
売り込みじゃなくて30分くらいZoomか電話で聞きたいだけ、とのこと。

もしよければ[自分の名前]に直接一声かけてみて。
（無理なら全然スルーでOK）
```

### 返信用（Layer C）— 紹介先から連絡が来たら

```
はじめまして／お久しぶりです、[自分の名前]です。

[紹介者の名前]から話聞いてもらえるって聞いて連絡しました。
中国語、勉強してる／現場で使ってる感じの話を30分くらい聞かせてほしいです。製品の営業とかはしません。

都合いい日時、教えてもらえますか？よければこちらで調整します： [予約URL]
```

## 2. ポーカーのときだけ会う知り合い

距離感が「友達」より遠いので、**依頼の輪郭**を最初からはっきり書く。

### A: 紹介を頼む（LINE）

```
[名前]、お疲れさまです。[自分の名前]です。

突然なんですが、中国語の勉強のリサーチをしていて、マカオ・海外ポーカーまわりの日本人に話を聞きたいです。
心当たりがいたら、1人だけ転送用の文を送るので、よければ転送してもらえませんか。

30分程度・営業はしません。無理なら気にせずスルーで大丈夫です。
```

### A: 紹介を頼む（X DM）

```
[名前]さん、お疲れさまです。

中国語学習のリサーチで、海外ポーカー勢に30分話聞きたいんですが、紹介いただけそうな人いたら1人だけお願いしたいです。転送文用意してます。営業なしです
```

### 転送用（Layer B）

```
ポーカーで知り合った[自分の名前]からのお願いです。

中国語の勉強（現場でどう使ってるか、アプリや勉強法）について、同じような境遇の人の話を集めているそうです。
30分程度、Zoomなどで。営業や勧誘はないとのことです。

よければ[自分の名前]へ連絡してみてください。
```

### 返信用（Layer C）

```
[自分の名前]です。転送ありがとうございました／お声がけありがとうございます。

中国語の勉強・現場での使用感について、30分ほどお話を伺いたいです。リサーチ目的で、サービスの販売などはしません。

ご都合のよい時間帯を教えていただくか、こちらから日程調整してもよければ：[予約URL]
```

## 3. 年上の人（先輩・年長の知人）

**LINE** は丁寧語。**X** も「です・ます」で統一し、絵文字は控えめに。

### A: 紹介を頼む（LINE）

```
[名前]さん

お世話になっております、[自分の名前]です。

一点お願いでご連絡しました。中国語の勉強について、現場で使われている方のお話を伺いたく、リサーチをしております。
マカオや海外で活動されている日本人の方、もしくは中国語を業務・生活で使われている方で、お心当たりがいらっしゃいましたら、お一人だけご紹介いただけますと幸いです。

所要は30分程度、営業はいたしません。転送用の文章をお送りしますので、コピペで構いません。お忙しいところ恐縮ですが、難しければ結構です。
```

### A: 紹介を頼む（X DM）

```
[名前]さん

お世話になっております、[自分の名前]です。
中国語学習のリサーチのため、現場で中国語を使われている方に30分ほどお話を伺いたいのですが、ご紹介可能な方がお一人でもいらっしゃいましたらお願いしたく存じます。営業はございません。転送用の文面をお送りします。
```

### 転送用（Layer B）

```
[紹介者の名前]のご友人の[自分の名前]よりご連絡です。

中国語の学習・実務での使用について、実際の経験を伺いたく取材（非営業・個人のリサーチ）をしております。オンラインで30分程度を想定しており、商品やサービスの勧誘はございません。

お心当たりがございましたら、[自分の名前]までご連絡いただけますと幸いです。
```

### 返信用（Layer C）

```
はじめまして、[自分の名前]と申します。

このたびはご連絡いただきありがとうございます。中国語の学習・ご利用状況について、30分程度お伺いできましたら助かります。営業目的のご案内はいたしません。

ご都合のよい日時をお知らせいただくか、下記よりお選びください：[予約URL]

どうぞよろしくお願いいたします。
```

## 本人に直接「話聞かせて」と頼む場合（Layer A 相当・紹介なし）

紹介ではなく、**その人に直接**打診するとき用。  
**具体化のコツ**: 「中国語の話」だけにせず、聞きたいことを2〜3個列挙すると返信率と当日の質が上がる（例: アプリと現場のギャップ、卓上・店頭で言えなかった経験、勉強のやり方の変遷など）。**問題インタビュー**なので製品名は出さなくてよい。

| 区分 | 案の目安 |
|------|-----------|
| 仲の良い友達 | 案1＝トピック列挙／案2＝感情・愚痴OK／案3＝日程先／案4＝X／案5＝超短く畳む／案6＝当日のお願い（紹介1人）まで先出し |
| ポーカー知り合い | 案1＝シーン列挙／案2＝負担軽め／案3＝マカオ前提／案4＝X／案5＝勉強なしでも可／案6＝対面カフェ |
| 年上 | 案1＝丁寧・列挙／案2＝短め／案3＝録音打診あり／案4＝X／案5＝録音なし明言／案6＝対面orオンライン選択 |

---

### 仲の良い友達に直接

#### 案1（トピック列挙・LINE）

```
[名前]、中国語の話30分くらい聞かせてほしい。

いま個人でリサーチしてて、具体的には
・DuolingoとかChatGPT、翻訳アプリ、何をどう使い分けてるか
・現場で「言いたいのに出てこなかった」系の話
・マカオとか海外で中国語触れる場面どうしてるか
あたりを雑談で聞きたい。Zoomでも電話でも。営業とか勧誘なし。

今週か来週で空いてる枠ある？
```

#### 案2（短め・感情寄り・LINE）

```
[名前]、中国語の勉強／現場で使う話、愚痴レベルでいいから30分聞かせて。

「アプリでは分かるのに口が追いつかない」とか「あのとき言えなかった」みたいな話がほしい。リサーチ用で売り込みゼロ。

都合いい日送って
```

#### 案3（日程ファースト・LINE）

```
[名前]、来週あたりで30分、中国語まわりの話聞かせてくれない？

勉強のやり方、現場で困ったこと、何使ってるか、くらいをメモ程度に聞くだけ。製品の話はしない。

火水木の夜とか、空いてるとこある？
```

#### 案4（X DM・2通想定）

**1通目**

```
[名前]、元気？中国語の勉強と現場での使い方、30分くらい話聞かせてほしい。Duolingo/翻訳/ChatGPTの使い分けと「言えなかった」話が聞きたい。リサーチで営業なし。今週来週で空いてる時間ある？
```

**2通目（返信後・URL送るとき）**

```
ありがとう。ここから選んでくれれば：[予約URL]　無理なら日時テキストでもOK
```

#### 案5（聞くことを一文で畳む・LINE）

```
[名前]、30分だけ中国語の話聞かせて。

「普段どう勉強してるか」「現場で言葉が詰まったときどうするか」「DuolingoとかAIとか翻訳、どれをどういうときに使うか」を聞きたい。メモ用で売り込みなし。今週来週で空いてるとこある？
```

#### 案6（当日のお願いまで先出し・LINE）

```
[名前]、中国語まわり30分インタビューさせて。

内容は質問いくつか用意してるけど脱線大歓迎。最後に「同じ悩みしてそうな人いれば1人だけ紹介して」ってお願いするかも。無理なら断ってOK。

Zoomで今週か来週、どっか取れる？
```

### ポーカー知り合いに直接（距離感ややフォーマル）

#### 案1（シーン明示・LINE）

```
[名前]さん、[自分の名前]です。

突然ですが、中国語まわりで30分ほどお話を伺いたいです。個人のリサーチで、内容は例えば
・マカオ／海外で中国語が出る場面（店・タクシー・卓外など）でどうしてるか
・勉強してる場合は、何を週何くらいやってるか
・「言いたかったのに言えなかった」があればそのときの対処
くらいです。Zoom希望ですが、電話でも構いません。勧誘や営業はしません。

今週〜来週でご都合いい時間帯、ありますか？
```

#### 案2（負担軽め・LINE）

```
[名前]さん、お疲れさまです。[自分の名前]です。

中国語、勉強中／現場で使う機会あります？あれば30分だけZoomで聞かせてほしいです。

アプリと現場のギャップとか、困ったエピソードとか、ざっくりでOK。中国語アプリのリサーチで、サービスの販売はしません。

返信つらければスルーで大丈夫です。空いてる曜日とか送ってもらえればこちら合わせます。
```

#### 案3（マカオ前提・LINE）

```
[名前]さん、[自分の名前]です。

マカオ滞在とか中国語の話、30分くらい聞かせてもらえませんか。
現地でどんな場面で中国語触るか、勉強してるなら何使ってるか、をメモしたいだけです。営業なし。

平日夜か、セッションじゃない日の昼とか、どっちが話しやすいですか？
```

#### 案4（X DM）

```
[名前]さん、[自分の名前]です。中国語（勉強／現場）の話30分Zoomで聞かせてほしいです。マカオで出る場面とアプリの使い分けとか。リサーチのみで勧誘なし。今週来週で都合いい枠あれば教えてください
```

#### 案5（勉強してなくてもOK・LINE）

```
[名前]さん、[自分の名前]です。

中国語、勉強してなくても大丈夫なんですが、マカオとかで店・移動・雑談で中国語が出る場面があれば、そこでどうしてるか（黙ってる／翻訳／単語だけ等）30分聞かせてほしいです。リサーチのみ、営業しません。

Zoomか電話、どちらが話しやすいですか？
```

#### 案6（対面カフェ希望・LINE）

```
[名前]さん、[自分の名前]です。

中国語の話、カフェで30分くらい聞かせてもらえませんか（オンラインでも可）。現場での使い方と、使ってるアプリ・ツールの話をメモしたいだけです。勧誘はしません。

◯◯駅周辺とか、行きやすい場所あれば教えてください。無理ならZoomにします。
```

### 年上の人に直接（です・ます）

#### 案1（トピック＋時間・LINE）

```
[名前]さん

お世話になっております、[自分の名前]です。

中国語の学習および実務・生活でのご利用について、30分程度お伺いできればと考えております。個人のリサーチ目的で、たとえば次のような点をお聞きしたいです。
・お使いの学習ツール（アプリ、教材、講師など）とその使い分け
・現場で表現に詰まった経験と、その後の対応
・（該当されれば）海外赴任・出張時の中国語の使われ方
商品やサービスのご案内はいたしません。

オンライン（Zoom等）を想定しております。ご都合のよい日時をお知らせいただけますでしょうか。
```

#### 案2（コンパクト・LINE）

```
[名前]さん、[自分の名前]です。

中国語の学習・ご利用状況について、30分ほどオンラインでお話を伺えませんでしょうか。使用されているアプリや教材、現場での困りごとを中心に伺いたいです。営業目的ではございません。

ご都合のよい候補日時を2〜3個いただけますと助かります。
```

#### 案3（録音の打診を軽く入れる版・LINE）

```
[名前]さん

[自分の名前]です。中国語の学習・実務での使用感について、30分程度インタビュー形式でお話を伺いたく存じます（個人リサーチ、勧誘はございません）。

内容は学習の進め方、ツールの選択、現場での言語使用の実感などを想定しております。可能であれば内容確認のため録音させていただきたいのですが、難しければ録音なしでも構いません。

Zoomにて、ご都合のよい日時をお聞きしてもよろしいでしょうか。
```

#### 案4（X DM）

```
[名前]さん、お世話になっております、[自分の名前]です。中国語の学習・ご利用について30分程度オンラインでお伺いしたく、個人でリサーチしております（営業はございません）。アプリ・教材のご利用状況と現場でのご体感を中心にお聞きしたいです。ご都合のよい日時をお知らせいただけますと幸いです。
```

#### 案5（録音なし前提・負担最小・LINE）

```
[名前]さん

[自分の名前]です。中国語の学習・ご利用について30分、オンラインでお伺いしたく存じます。録音はいたしません（メモのみ）。伺う内容は、ご利用のツールと、現場で表現に困ったときの対処が中心です。営業はございません。

ご都合のよい日時を1ついただければこちらで候補をお送りします。
```

#### 案6（対面／オンライン選択式・LINE）

```
[名前]さん

お世話になっております、[自分の名前]です。

中国語の学習・実務でのご使用について30分程度お伺いしたく、個人のリサーチとして実施しております。オンライン（Zoom等）に加え、ご都合が合えばカフェ等での対面も可能です。商品・サービスのご案内はございません。

ご希望の形式（オンライン／対面）と、ご都合のよい曜日・時間帯をお知らせいただけますでしょうか。
```
