# Phrabit ロードマップ 2026-2028

**作成日**: 2026-06-01  
**参照**: `docs/strategy-reset-2026.md`, `docs/market-competitor-refresh-2026.md`  
**目的**: 英中対応後のPhrabitについて、3ヶ月・6ヶ月・12ヶ月・24ヶ月の目標と、拡張/課金判断の条件を定義する。

---

## 0. 基本方針

Phrabit は、最初から大きな語学学習アプリとして広げない。

まず海外ポーカーで、次の仮説を検証する。

> 翻訳した一言を保存し、次に自分で言えるように復習することに価値があるか。

その価値が見えたら、海外旅行・長期滞在・出張へ広げる。
言語は中国語・英語を起点にし、将来的には他言語へ拡張できる状態を維持する。

## 2. Phase 1: 告知・初期利用観察

### 期間

リリース告知後、最初の1ヶ月。

### 目的

多くの機能を足すことではなく、初めて使う人が何でつまずくかを見る。

### 主要ターゲット

- 海外ポーカーをする日本人
- WSOPや海外遠征に関心がある日本人
- マカオ・ラスベガスなどで英語/中国語に少し困る人

### やること

- Xで告知する
- 知人・現地プレイヤーに使ってもらう
- スマホ実機で利用観察する
- 使われた言語、保存、ドリル追加、離脱理由を見る
- 使い方が分からない箇所を直す

### 追う指標

| 指標 | 目安 |
|---|---:|
| 初回利用者 | 50〜150人 |
| 翻訳実行者 | 30人以上 |
| 2回以上使った人 | 10人以上 |
| 明確なフィードバック | 10件以上 |
| 保存またはドリル追加した人 | 5〜10人以上 |

### 判断

次のどれかを判断する。

| 結果 | 判断 |
|---|---|
| 翻訳だけ使われ、保存/ドリルがほぼ使われない | 差別化が弱い。保存導線か価値訴求を見直す |
| 保存はされるがドリルされない | ドリルが重い。復習導線を軽くする |
| 英語利用が多い | 英語対応を前面に出す余地あり |
| 中国語利用が多い | 中国語・マカオ文脈を継続 |
| 速度不満が多い | 速度改善を最優先 |

---

## 3. Phase 2: 顧客実証

### 期間

1〜3ヶ月。

### 目的

「便利そう」ではなく、実際に繰り返し使われるかを見る。

### やること

- 保存/ドリル追加の導線を改善する
- 英語・中国語の切替で迷わないUIにする
- 追加フレーズ生成の品質を改善する
- 実際に使った場面をログ化する
- 使わなかった人にも理由を聞く

### 追う指標

| 指標 | 目安 |
|---|---:|
| 月間利用者 | 100〜300人 |
| 週2回以上使う人 | 10〜30人 |
| 保存/ドリル追加率 | 翻訳実行者の20%以上 |
| D1再訪 | 20%以上 |
| D7再訪 | 10%以上 |
| 「Google翻訳ではなく使う理由」を言える人 | 10人以上 |

### 判断

Phase 2終了時点で、次の問いに答える。

1. Phrabitを開く理由は、翻訳以外にあるか
2. 保存・復習は本当に使われているか
3. 英語と中国語で利用行動に差があるか
4. ポーカー以外の場面が自然に出ているか

### 継続条件

以下を満たすなら、課金価値検証へ進む。

- 10人以上が複数回使っている
- 保存またはドリル追加が自然に起きている
- 3人以上が「これなら有料でも考える」と言う
- 使わない理由が修正可能なUX問題に寄っている

## 4. Phase 3: 課金価値検証

### 期間

3〜6ヶ月。

### 目的

マネタイズ方法を決める。
ただし、いきなり大規模課金ではなく、小さく検証する。

### 課金候補

| 方式 | 仮説 | 初期判断 |
|---|---|---|
| 月額 | 継続利用者に合う | 本命候補 |
| 遠征前買い切り | WSOP/旅行前に合う | 検証価値あり |
| フリーミアム | 無料翻訳 + 有料保存/生成/同期 | 初期に一番自然 |
| AI教師 | 自分の弱点や目的に合わせる | 中期候補 |
| フレーズパック | ポーカー/旅行/出張別 | 軽い課金候補 |
| 広告 | 無料利用者から収益化 | 初期は非推奨 |

### やること

- 課金インタビューを入れる
- 有料プランの仮ページまたは待機リストを作る
- 「有料なら欲しい機能」を聞く
- 価格を複数案で聞く
- 可能なら手動で有料βを試す

### 価格仮説

| プラン | 仮価格 | 想定 |
|---|---:|---|
| 月額ライト | 480〜980円 | 個人の継続利用 |
| 月額プロ | 1,480〜2,980円 | AI生成・同期・高頻度利用 |
| 遠征パック | 980〜2,980円 | WSOP/マカオ前の買い切り |
| フレーズパック | 300〜980円 | テーマ別の追加コンテンツ |

### 判断

次のどれかを決める。

| 結果 | 判断 |
|---|---|
| 月額に反応がある | サブスクβへ |
| 遠征前買い切りに反応がある | パック型課金へ |
| 課金反応が弱いが利用はある | 無料利用を伸ばして後で課金 |
| 利用も課金反応も弱い | セグメントまたは価値仮説を見直す |

## 5. Phase 4: セグメント拡張

### 期間

6〜12ヶ月。

### 目的

海外ポーカーで見えた勝ち筋が、旅行・長期滞在・出張に広がるかを検証する。

### 拡張候補

| 候補 | 優先度 | 理由 |
|---|---|---|
| 海外旅行者 | 高 | 英語対応と相性が良い。母数が大きい |
| WSOP/ラスベガス遠征 | 高 | 既存ポーカー文脈から近い |
| マカオ/アジアカジノ | 中 | 中国語・英語両方が出る |
| 駐在・長期滞在 | 中 | 継続利用しやすいが品質要求が高い |
| ビジネス出張 | 中〜低 | 支払い意思は高いが誤訳リスクが重い |
| 語学学習者全般 | 低 | 競合が強い。後回し |

### やること

- 場面別の入口を作る
- ポーカー以外の具体状況入力を強化する
- 自分のプロフィール/目的からフレーズ生成する機能を検討する
- SEOやX投稿で「海外旅行 英語 フレーズ 保存」などの切り口を試す

### 判断

次を満たせば、ポーカー外へ正式拡張する。

- ポーカー外ユーザーが自然に発生している
- 保存・ドリル追加率がポーカー層と大きく落ちない
- 「旅行/滞在でも使う理由」が明確に言語化できる
- UIが複雑にならない

---

## 6. Phase 5: 収益拡大

### 期間

12〜24ヶ月。

### 目的

検証済みの勝ち筋を、収益に変える。

### 到達目標

| 時期 | ユーザー目標 | 課金目標 | 売上目安 |
|---|---:|---:|---:|
| 12ヶ月 | 月間利用 500〜1,500人 | 有料 50〜200人 | 月5万〜30万円 |
| 24ヶ月 | 月間利用 3,000〜10,000人 | 有料 300〜1,500人 | 月30万〜200万円 |

この数字は目標であり、予測ではない。
Phase 1〜3の実データで更新する。

### 拡張候補

- 多言語追加
- パーソナルプロフィールからのフレーズ生成
- 場面別フレーズパック
- 有料同期
- 高品質音声
- 学習履歴・弱点分析
- チーム/法人向け管理

### やらないこと

- 競合の語学アプリを丸ごと追いかける
- 教材カリキュラムを大量制作する
- 広告でUXを壊す
- 使われていない機能を大きく作り込む

---

## 7. 拡張判断ルール

### 言語追加

追加してよい条件:

- 既存の英語/中国語で保存・復習価値が確認できている
- その言語を使う具体セグメントがある
- 翻訳品質・音声・解説の最低品質を担保できる
- UIが複雑にならない

追加しない条件:

- ただ「多言語対応っぽい」だけ
- 利用者の具体ニーズがない
- 言語追加により既存UXが分かりにくくなる

### セグメント拡張

拡張してよい条件:

- 既存セグメントで10〜30人の継続利用がある
- 拡張先ユーザーが、同じ課題を別文脈で持っている
- 既存機能を大きく変えずに使える

拡張しない条件:

- 既存セグメントの価値が未確認
- 広げるほどメッセージがぼやける
- 競合と正面衝突するだけになる

### 課金開始

開始してよい条件:

- 複数回使う人がいる
- 保存/ドリル/生成のどれかに明確な価値がある
- 価格質問で拒否だけでなく比較・条件付きYESが出る
- 手動課金でも数人が払う可能性がある

開始しない条件:

- 初回利用だけで終わっている
- 保存も復習も使われていない
- 「無料なら使う」しか出ていない

## 8. 直近1ヶ月の実行リスト

1. Xで告知する。
2. 本番URLでスマホ実機テストをする。
3. Supabase/Notionのテストデータを告知前に確認する。
4. 利用者に「何に使ったか」「使わなかった理由」を聞く。
5. 英語/中国語の利用比率を見る。
6. 保存・ドリル追加率を見る。
7. 速度不満を記録する。
8. 課金質問を軽く入れる。

聞くべき質問:

- どの場面で使いましたか。
- Google翻訳やChatGPTではなく、これを開く理由はありましたか。
- 保存やドリル追加は使いましたか。
- 使わなかったなら、なぜですか。
- 有料なら、どの機能に払ってもよいと思いますか。
- 月額と遠征前買い切りなら、どちらが自然ですか。

---

## 9. 最重要判断

今後3ヶ月で見るべき最重要判断はこれ。

> Phrabit は、翻訳アプリではなく「自分の現場フレーズを育てる道具」として使われるか。

これがYESなら、ポーカー外へ広げる価値がある。
NOなら、翻訳速度・保存導線・ドリル体験・ターゲットのどれかを見直す。
