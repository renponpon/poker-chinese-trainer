## 週次チェックリスト（コピー用）

### 今週（Week N）

- [ ] **インタビュー** ___ 件実施（目標: まず週2〜）
- [ ] **観察シート** を直後3分以内に記入
- [ ] **紹介依頼**「同じ悩みの人を1人だけ」___ 件（**口頭CLOSE**で可。一括DMは現方針では送らない → [`phase2-open-todos.md`](./phase2-open-todos.md)）
- [ ] **X公募** — **一旦保留**（ビーチヘッド30人以上あり。母数不足・断られ続けるときだけ）[`x-recruitment-draft.md`](./x-recruitment-draft.md)
- [ ] **ご意見番** DM ___ 通（[`opinion-leader-candidates.md`](./opinion-leader-candidates.md)）
- [ ] `persona-draft.md` の **テスター欄** を1行でも更新
- [ ] 気づきを `customer-discovery.md` の該当仮説に **1箇所だけ** メモ追記（長文化しない）

### インタビューが5件溜まったら

- [ ] 痛みワードを一覧化（コピペでスプレッドシートでも可）
- [ ] 「右上ポジション」の説明が**刺さる／刺さらない**を判定
- [ ] 次の週の **打診優先セグメント** を1行で決める
