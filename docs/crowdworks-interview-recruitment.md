# クラウドワークス：非ビーチヘッド向けインタビュー募集ブリーフ

**前提（なぜ絞るか）**: 検証したいのは「**日本語話者として中国語を使う現場で、言葉が反射的に出ない・損・ストレスが出る**」という課題の実在なので、**すでに母語級・準母語級で現場が楽な人**の話は**別問題**になる。CWではその層を**募集段階で外す**。

---

## 1. 適正メンバー（採用したい人）

### スイートスポット（一文）

**日本語が思考・交渉の主言語**であり、中国語圏では **「聞き取り・話す・場の空気」でまだ詰まる・準備しないと不安」** を**最近も**経験している人。

### 案件A：中国出張ビジネス層（優先）

| 項目 | 条件 |
|------|------|
| **必須** | 日本在住または日本法人所属で、**直近12か月以内に業務で中国語圏（大陸・香港・台湾含むかは募集で明示）に渡航した** |
| **必須** | 現地で **口頭で中国語が出る場面** があった（会議・工場・接待・税関・移動など何でも可） |
| **必須** | **オンラインで30分**、カメラオンまたは音声のみ可、**個人開発のリサーチ**に協力できる |
| **必須** | **母語は日本語**（下記「境界」のとおり。中国語母語・日本育ちの二言語話者は**別枠扱い**で基本除外） |
| **歓迎** | 中国語は「完璧ではない」「現場で言いたいことが出てこない」感覚がある |
| **歓迎** | Duolingo・ChatGPT・翻訳アプリ・教室のいずれかを使った／使わなかった理由が言語化できる |

**適正の例**: 商社・メーカー営業・購買・設計駆けつけ・スタートアップの海外担当・バイヤー等で、年数回以上の渡航がある人。

### 案件B：駐在・長期赴任経験者

| 項目 | 条件 |
|------|------|
| **必須** | **過去5年以内に3か月以上** 中国語圏に駐在・長期出張していた |
| **必須** | 業務または日常生活で中国語に触れていた |
| **必須** | **母語は日本語**（同上） |
| **必須** | 同上オンライン30分・リサーチ協力 |
| **推奨（境界対策）** | **累計滞在が極端に長い（例: 中国語圏で通算5年以上）**で、すでに「業務は中国語でほぼ問題ない」と感じている人は**本募集の適正から外れる**ことが多い（下記「非適正」参照） |

**適正の例**: 現地法人勤務・プロジェクト赴任・家族帯同駐在など、**まだ日本語が仕事の土台**で、現場中国語にギャップを感じていた時期がある人。

### 境界（迷ったとき）

| ケース | 扱い |
|--------|------|
| **中国語母語**（中国人・シンガポール華人など）で日本語も可 | **本募集では不適正**（課題構造が違う） |
| **日本国籍の帰国子女**で中国語が事実上ネイティブ | **原則除外**。どうしても聞くなら別枠・別仮説 |
| **滞在5年以上だが、まだ現場で毎回詰まる** | **適正になりうる**（「ネイティブチック」かどうかは**自己申告ではなくエピソード**で見る） |

## 2. 不適正・除外したい人

### 明確に除外（募集文に書けるもの）

- **中国語が母語**、または **幼少から家庭・学校で中国語が主**で、ビジネス中国語が**日本語並みに扱える**方  
- **中国語圏の累計滞在が長く**（目安 **通算5年以上**）、かつ **業務で中国語に困ることがほぼない**と感じている方（＝いわゆるネイティブチック層。長期でも**まだ困る**なら対象外ではない）  
- **マカオ等でポーカーが主目的の滞在**が中心の人（ビーチヘッドは別ルートで十分。**「カジノ目的の渡航は対象外」**と明記）  
- **本業が日中通訳・翻訳**の人（一般ユーザーと課題の感じ方が違う。**別枠**にするか募集から外す）

### 運用上の除外

- 応募文がテンプレのみで、**業種・渡航時期・場面の具体がゼロ**の人（作り話・サクッと稼ぎ目的のリスク）

## 3. スクリーニング（応募時に必ず書かせる）

応募フォーム or 最初のメッセージで **自由記述** を必須にする（Yes/No だけだと通りやすい）。

1. **直近の中国語圏渡航**はいつ・何日程度・どの都市か（複数可）  
2. **業務内容**（職種・業界は2行で可）  
3. **現地で中国語を使った具体例を1つ**（誰と・どこで・何が伝わりにくかったか、でも可）  
4. **中国語学習**（今やっていること／やめた理由。やっていなければ「していない」で可）  
5. **母語・幼少の主言語**（日本語／中国語／両方 等、一言で可）  
6. **中国語圏の累計滞在**（おおよそで可：〇年〇か月、または「ほぼなし」）  
7. **業務で中国語を使うとき**、「ほぼ問題なくできる」「たまに詰まる」「たまに困る」「よく困る」から近いものを1つ選ぶ  
8. **オンライン面談可能な曜日帯**（3つ）

**採用判断**: 1〜4が具体的であること。**5が中国語母語かつビジネスも中国語中心**なら原則落とす。**6が通算5年以上**なら、7が「ほぼ問題なく」に寄っていないか確認。迷ったら **「最近、言いたかったのに出てこなかった場面」** を一文で書かせて、具体がなければ見送り。**日本語で問題ない**旨も書いておく。

## 4. 募集タイトル案（コピペ用）

**案件A用**

```
【謝礼3,000円／30分】中国語圏への業務渡航経験者限定｜中国語の困りごとオンラインヒアリング（個人開発リサーチ）
```

**案件B用**

```
【謝礼3,000円／30分】中国語圏に3か月以上駐在・長期赴任した経験者｜オンラインヒアリング（個人開発リサーチ）
```

謝礼は **3,000〜5,000円** の範囲で調整可（[`customer-discovery.md`](./customer-discovery.md) 戦術5）。反応が薄ければ **4,000〜5,000円** に上げる。

## 5. 募集本文テンプレ（プロジェクト説明）

案件Aをベースに。**{}** は差し替え。

```
【内容】
個人で中国語の学習ツールを開発しており、設計の参考として「業務で中国語圏に渡航したときの困りごと」を伺いたいです。
営業・勧誘は一切ありません。課題のヒアリングのみです。

【条件】
・直近12か月以内に、業務で中国語圏（{大陸／香港を含むか等}）へ渡航したこと
・現地で中国語を口頭で使う（または使おうとした）場面があること
・思考・交渉の主言語が日本語であること（中国語母語の方は対象外）
・Zoom または Google Meet で30分程度オンライン対応可能なこと
・日本語で問題ないこと

【対象外】
・中国語が母語、または幼少から中国語が主言語で、業務も中国語でほぼ問題なく行える方
・中国語圏への累計滞在が長く（目安として通算5年以上）、すでに業務中国語で困ることがほぼない方（ネイティブに近い運用の方）
・本業が日中通訳・翻訳の方（別募集でお願いする可能性があります）
・カジノ・ポーカーが主目的の渡航のみの方

【報酬】
{3,000}円（税込の扱いはCWの慣行に合わせる）／1名あたり、インタビュー実施後にお支払い

【応募時に必ず記載ください】
（上記スクリーニング1〜5をそのまま貼る）

【実施後】
内容は開発の参考にのみ使用し、個人を特定する形で公開しません。録音する場合は当日冒頭で改めてご説明し、同意いただきます。
```

## 6. 募集の分け方のおすすめ

| 方針 | メリット |
|------|----------|
| **案件AとBを別プロジェクトに分ける** | 応募が混ざらず、スクリーニングが楽 |
| **まず案件Aのみで5名** | 第1拡張の仮説に直結 |
| **必要なら案件Bを追加** | 駐在の深い話は母数が取りにくいので単価微増も検討 |

## 8. 適正人数の目安

非ビーチヘッドの **深いインタビュー** は、セグメントあたり **3〜5人** あれば判断材料になりやすい（会話で合意した「合計6〜8人」の内数）。  
クラウドワークスは **応募多・質ばらつき** のため、**採用5・実施3** を見込んで募集人数を決めるとよい。
