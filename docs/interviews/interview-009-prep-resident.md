## 1. このインタビューで特に見たいこと

今回の駐在層が、既存サンプルのどちらに近いかを判定する。

- **健太さん型**: 低学習意欲・翻訳依存・会話補助ニーズが中心
- **森様型**: 高学習意欲・生活リスニング課題・翻訳履歴ドリル化ニーズが中心
- **古屋様型**: 高習熟・実務ニュアンス・場面適合翻訳ニーズが中心
- **別タイプ**: 既存3類型に収まらない駐在層

最重要の判定軸は、習熟度ではなく **学習動機と雇用したい解決策**。

## 2. 必ず回収したい未検証ポイント

### A. 中国語→日本語方向の頻度

確認したいこと:

- 相手の中国語を聞き取れず困る場面がどれくらいあるか
- 会話中に翻訳アプリを開くか
- 「言いたい」より「聞きたい/分かりたい」が強いか

聞き方:

```
最近、中国語で一番困った場面を1つだけ思い出すと、相手の中国語が分からなかった場面ですか？
それとも、自分が中国語で言えなかった場面ですか？
```

追加:

```
相手の中国語をその場で翻訳したいと思う場面は、1週間でどれくらいありますか？
```

### B. 会話中にアプリを開く心理的ハードル

確認したいこと:

- 専用アプリを会話中に開くのは自然か
- Google翻訳より遅い場合でも、保存/復習価値で使われるか
- 人前で音声入力や音声再生をする抵抗があるか

聞き方:

```
会話中にGoogle翻訳や翻訳アプリを開くことはありますか？
ある場合、相手を待たせる感じや気まずさはありますか？
```

追加:

```
もし翻訳した内容が後から自動で見返せる/復習できるとしても、
Google翻訳より少し手間が増えたら使わなくなりそうですか？
```

### C. 翻訳ログを「覚えたい」か「参照したい」か

確認したいこと:

- 翻訳した内容を後で覚えたいか
- 単に辞書/履歴として見返せれば十分か
- 全件保存とドリル対象選択のどちらが自然か

聞き方:

```
翻訳アプリで調べた表現を、後から見返すことはありますか？
見返すとしたら「覚えたい」感じですか、それとも「また必要な時に探せればいい」感じですか？
```

追加:

```
翻訳したものが全部ドリルに入ると、便利そうですか？
それとも、覚えたいものだけ自分で選べる方がよさそうですか？
```

### D. 保存・プライバシーの許容ライン

確認したいこと:

- 会話内容の保存に抵抗があるか
- 業務/個人/家族/病院などで保存したくない領域があるか
- 「保存する/しない」を選べれば十分か

聞き方:

```
翻訳した内容がライブラリに保存されるとしたら、抵抗はありますか？
仕事・家族・病院・お金の話などで、保存したくない内容はありますか？
```

追加:

```
毎回保存する/しないを選べる、または後から消せるなら安心感は変わりますか？
```

### E. 状況別フォルダ/タグの分類軸

確認したいこと:

- 駐在生活で「状況別に見返したい」ニーズがあるか
- どの分類が自然か
- 自動分類と手動フォルダ、どちらがよいか

聞き方:

```
もし自分が保存した中国語フレーズが溜まっていくとしたら、
どういう分類で見返せると便利そうですか？
```

候補を出す場合:

```
仕事、レストラン、病院、子ども関連、買い物、移動、住居トラブル、友人との会話などだと、
どれが一番使いそうですか？
```

### F. 支払い意思

確認したいこと:

- 月額/買い切り/無料枠の感覚
- 個人課金か経費か
- 課金トリガーは会話補助か学習か

聞き方:

```
もしこの困りごとがかなり解決するとしたら、個人で月いくらくらいまでなら払ってもいいと思いますか？
無料なら使うけど有料なら使わない、月500円ならあり、月1500円でもあり、など感覚で大丈夫です。
```

追加:

```
仕事で使う場合、経費で払える可能性はありますか？
それとも完全に個人利用の感覚ですか？
```

## 3. 30分の進行案

### 0:00-0:03 OPEN

- 目的説明
- 売り込みではなく、困りごとの理解が目的と伝える
- 録音/メモ確認

### 0:03-0:08 SITUATION

聞くこと:

- 中国での生活/仕事の状況
- 1日の中で中国語に触れる場面
- 日本語/英語/中国語の使い分け

優先質問:

```
1日の中で、中国語を使う/聞く場面はどこにありますか？
仕事・生活・家族・買い物など、頻度が高い順に教えてください。
```

### 0:08-0:16 WARM/DEEP

聞くこと:

- 直近で困った具体場面
- その時本当は何を達成したかったか
- 何が邪魔したか

優先質問:

```
最近、中国語で「これは困った」「面倒だった」「悔しかった」と感じた場面を1つだけ、
できるだけ具体的に教えてください。
```

深掘り:

```
その時、本当はどうなってほしかったですか？
その場ではどう対処しましたか？
後から見返したり、覚えようとしたりしましたか？
```

### 0:16-0:24 CHECK

聞くこと:

- 既存解
- 翻訳/AI
- 保存/復習
- 状況別整理
- プライバシー

優先質問:

```
普段、中国語で困った時は何を使っていますか？
Google翻訳、DeepL、WeChat翻訳、ChatGPT、中国系AI、周囲の人に聞く、英語で押す、など。
```

次に:

```
そのやり方で、一番面倒なことや不安なことは何ですか？
```

### 0:24-0:28 コンセプト軽確認

プロトを見せる場合も、最後に短く。

聞くこと:

```
翻訳した/困った中国語がライブラリに残って、必要なものだけ状況別に整理したり、
ドリル化できるとしたら、どの部分が一番使えそうですか？
逆に、これはいらない/重いと感じる部分はありますか？
```

### 0:28-0:30 CLOSE

必ず聞く:

```
同じように中国語で困っている駐在員・帯同家族・出張者で、
1人だけお話を聞けそうな方はいますか？
```

余裕があれば:

```
もし使うとしたら、無料なら使う/有料でも使う/どのくらいなら払う、の感覚はありますか？
```
