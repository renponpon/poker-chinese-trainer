# 1. 中国語学習の「底辺人口」プロキシ（試験受験者）

## 1-1. HSK（日本国内・年間受験者数）

| 年 | 年間受験者数（日本） | 出典・ラベル |
|----|---------------------|--------------|
| 2015 | 23,426人超 → **25,000人突破**を記念キャンペーン | [AT Press（HSK日本実施委員会）](https://www.atpress.ne.jp/news/69465) **A** |
| 2018 | **34,018人** | ブログがHSK公式サイトの推移を引用 [sadominhe.com](https://sadominhe.com/%E4%B8%AD%E5%9B%BD%E8%AA%9E%E6%A4%9C%E5%AE%9A%E3%81%A8hsk%E3%81%AE%E5%8F%97%E9%A8%93%E8%80%85%E6%95%B0%E3%81%AF%E5%A2%97%E3%81%88%E3%81%A6%E3%81%84%E3%82%8B%E3%81%AE%E3%81%8B%EF%BC%9F/) → 原本 [HSK日本 hskj.jp/about](https://www.hskj.jp/about/) **B→A** |
| 2021 | **39,219人**（年12回実施前提の年間合計） | 同上ブログ内表（出典HSK公式サイト） **B→A** |

**読み取り**

- 「年間**数万〜4万人前後**」が日本のHSK受験規模のオーダー感。
- コロナで一時減少後も、HSKは**中検より年間受験者総数が多い**という整理が複数ソースで一致（sadominhe 等）。
- **全学習者数ではない**（受験しない学習者・アプリのみ層は含まない）。

## 1-2. 中国語検定（中検）

- 2021年: 年間 **23,709人**（年3回） / 1回あたり平均 **7,903人**  
  出典: [sadominhe.com 同上記事](https://sadominhe.com/%E4%B8%AD%E5%9B%BD%E8%AA%9E%E6%A4%9C%E5%AE%9A%E3%81%A8hsk%E3%81%AE%E5%8F%97%E9%A8%93%E8%80%85%E6%95%B0%E3%81%AF%E5%A2%97%E3%81%88%E3%81%A6%E3%81%84%E3%82%8B%E3%81%8B%EF%BC%9F/) **B**

## 1-3. 試験受験者を合わせた「真面目層」のざっくり上限

- **HSK + 中検**を足すと二重カウントの可能性があるため、単純合算はしない。
- 保守的に見ても、**年間に資格・検定の形で中国語を測ろうとする人は数万人規模**（日本国内）。

**プロダクト示唆**

- 「HSKを受けるほど真面目だが、アプリでは続かない」層が一定数いる仮説と整合。
- インタビューで「HSK/中検を受けたか」を聞くと、**学習コミット度**のプロキシになる。

# 3. マカオと日本人（ビーチヘッド周辺）

## 3-1. 日本人のマカオ訪問者数（観光・短期の上限感）

報道ベース（**B**、数値はメディア二次引用のため本番資料では観光局一次を確認推奨）:

- 2024年通期、日本からのマカオ訪問者 **約126,424人**（前年比約68%増などの文脈）  
  例: [トラベルビジョン](https://www.travelvision.jp/news/detail/news-116193) **B**
- 別報道では2024年1〜11月 **114,314人** など表記差あり → **定義（国籍・延べ／人年）を要確認**

**読み取り**

- 「年に一度マカオに行く日本人」は**十万人未満〜十万人弱**オーダー。
- その中の**ポーカープレイヤー**はさらに極小（後述）。

## 3-2. マカオ日本人ポーカープレイヤー（プロジェクト内仮置き）

- **全体で約60人**、知人ネットワーク **約30人**（ユーザー提供・**C**）
- 事業のTAMではなく、**CPF検証用ビーチヘッド**の規模感として記録。

# 4. 日本国内のポーカー「人口」（参考）

## 4-1. メディア・業界記事ベース

- 「日本のポーカー人口は**100万人超**」などの表現が業界メディアに存在（定義は「一度でも触れた人」か不明）。  
  例: [カジノジャパン® 解説記事](https://casinojapan-inc.jp/feature-japan-poker/) **C**
- 国内アミューズメント系ポーカー店舗 **400店超** などの記述あり（同上系） **C**
- **JOPT** 大型イベントで**延べ5万人規模**の来場・動員などの報道  
  例: [Japan Open Poker Tour レポート](https://japanopenpoker.com/jopt2025gf_afterreport/) **B**

**読み取り**

- **ライブで中国語圏と毎週接するプロ／準プロ**は、上記「100万人」とは別次元の極小集合。
- 数字は**マーケ用に使うほど硬くない**ため、インタビューと併用して自分のセグメント定義を固める。

# 6. TAM / SAM / SOM（仮置き・要検証）

ここからは**算数**であり、**事実ではない**。インタビュー後に更新する。

| レイヤー | 定義（仮） | オーダー感 | 根拠の例 |
|----------|------------|------------|----------|
| **TAM** | 「中国語を仕事・生活で使う可能性がある日本人」 | **数十万〜数百万人** | 在留・出張・観光・製造サプライチェーン等、境界が曖昧 |
| **SAM** | 「中国語で現場ストレスがあり、スマホで学習する層」 | **数万人〜数十万人** | HSK+中検の年間受験者が**下限のヒント**（全学習者ではない） |
| **SOM** | 「課金して継続する」初期現実味 | **数百〜数千人** | ニッチ立ち上げの一般的射程。1480円×1000人で月約148万円 |

**プロダクトオーナー過去仮説との照合**

- `customer-discovery.md` にある「3年で課金5,000〜10,000人」は、上記SOMと**矛盾しないが未検証**。インタビューで支払い意思を更新すること。
