# 課題ヒアリング プレイブック

**オンライン 30分 | 資料を見せずに会話で進める**

---

## 🎯 目的

「中国語の課題仮説」が **実在し、深刻で、頻発しているか** を、相手の言葉と表情から検証する。

加えて、**JTBD（ジョブ理論）の観点**から以下を解像度高く把握する。

- 顧客が**進歩を遂げたい状況**は何か
- その状況で**今は何を雇用しているか**（既存解・翻訳・我慢・無消費）
- **どんな進歩を遂げたいか**、それを**阻む障害**は何か
- より良い解決策の **品質の定義**、引き換えに **譲ってもいいもの**
- 機能的／感情的／社会的の **3側面**
- 顧客が **したくないこと**

> 売らない。聞く。痛みワードと「状況」を拾う。

---

## 🧭 心構え（3つ）

1. **仮説を「壊しに行く」姿勢で。** 反証を歓迎する。同意だけ集めると検証にならない。
2. **沈黙に耐える。** 5秒待つと相手のベストアンサーが出る。質問を重ねて埋めない。
3. **メモは記号化。** フルセンテンスは録音任せ。会話に集中する。

---

## 🧠 JTBDレンズ（聞きながら頭の中で組み立てるもの）

| 観点 | 何を拾うか |
|------|----------|
| **状況（Situation）** | いつ・どこで・誰と・何をしているとき発生したか |
| **進歩（Progress）** | その状況で「本当は何を成し遂げたかった」のか |
| **障害（Obstacle）** | 進歩を阻んでいるもの（言葉・知識・場の空気・道具） |
| **雇用中の解決策** | いま何を雇って間に合わせているか（翻訳・英語・笑顔・諦め・自作ツール等） |
| **不満／我慢** | 今の解決策に対する具体的な不満・埋め合わせ行動 |
| **したくないこと** | 現状や既存解の中で **避けている／嫌だと思っている** 行動・体験。例：翻訳の待ち時間で相手を待たせる／自分に関係ない単語を覚える／翻訳機を相手に渡す／現地人にいちいち聞く／教室に通って予定を取られる |
| **品質定義** | より良い解決策の「これがあれば合格」と言える条件 |
| **引き換え** | 払えるお金・時間・手間・プライバシー等の対価 |
| **3側面** | 機能（できる／速い）／感情（恥・悔しさ・安心）／社会（評価・関係性） |

## ⏱ Timeline（30分）

| 時間 | フェーズ | やること |
|------|---------|---------|
| 0:00 – 0:03 | **OPEN** | 自己紹介・趣旨・録音許可 |
| 0:03 – 0:07 | **SITUATION** | 中国語に触れる状況・頻度 |
| 0:07 – 0:14 | **WARM / PROGRESS** | 直近エピソード＋本当は何をしたかったか |
| 0:14 – 0:21 | **DEEP** | 痛み・障害・3側面・損失 |
| 0:21 – 0:27 | **CHECK** | 翻訳/GPT等の雇用状況・品質定義・属性別追加質問 |
| 0:27 – 0:30 | **CLOSE** | 解決後の変化・引き換え・紹介依頼 |

> 30分で収めるため、**全員必須は太字質問だけ**。属性別質問は最大2問まで。機能細部（使用済ステータス、音声再生の可否など）は、相手がプロト試用者で時間が余った時だけ聞く。

## 🎤 質問テンプレ

> 読み上げNG。流れの呼び水として、自然な口調に変換して使う。

### OPEN（0:00 – 0:03）

> 「今日はお時間ありがとうございます。30分で、中国語に困った経験と、いま何で乗り切っているかについて伺いたいと思っています。**売り込みは一切なく**、純粋に体験談を聞かせてください。記録のために**録音させていただいてもよろしいですか？**」

### SITUATION（0:03 – 0:07）— 状況の現況

> 「進歩を遂げたい状況」を把握するための土台。`どこで・誰と・何のために中国語に触れているか` を立体的に。

- **差し支えない範囲で、いまどこで・どんな生活／仕事をされていますか？**
- **1日／1週間で、中国語に触れる場面はどのくらいありますか？**（話す・聞く・読む・書く）
- **いちばん中国語が必要になるのは、どんな場面ですか？**
- 余裕があれば：逆に、中国語を使わずに済ませている場面は？（**＝無消費・回避**）
　中国語の勉強はしていますか？している場合、何を使って勉強していますか？

### WARM / PROGRESS（0:07 – 0:14）— 直近の困り＋本当は何をしたかったか

> **痛みエピソード**と**遂げたい進歩**をセットで拾う。

- 最近、中国語が出てこなくて困った場面はありましたか？
- 具体的にどんな場面でしたか？（誰と／どこで／何を）
- どれくらいの頻度で、そういう瞬間がありますか？
- **その時、本当は何をしたかったんですか？**（買い物／指示／雑談に入る／意思を通す／関係を作る、等）
- **何が邪魔をしましたか？**（言葉が出ない／発音が伝わらない／相手のスピード／その場の空気、等）

### DEEP（0:14 – 0:21）— 痛み・3側面・損失

- その時、どう対処しましたか？
- 結果どうなりましたか？
- 終わった後、どんな気持ちが残りましたか？（**感情側面**）
- 周りの目はどう感じましたか／関係性に影響はありましたか？（**社会側面**）
- 機能的に何ができなかった／遅れた／失敗したと感じますか？（**機能側面**）
- これまでで一番痛かった経験は、10点満点で何点ですか？
- 損したものはありますか？（時間／お金／関係性／自尊心／機会）
- 同じような場面でも「**まあええか**」で流したことはありますか？ どんな時に流して、どんな時に流せませんでしたか？

### CHECK（0:21 – 0:27）— いま何を雇用しているか／品質定義

> ここは **「会話補助」と「学習資産化」が同じ場面で起きるか** を見る最重要パート。既存解の中身を「どう開いて／何分／何ステップ／誰と」まで聞く。

#### A. 全員に聞く（必須）

- **翻訳アプリやAI（ChatGPT等）を、会話や意思疎通のために開く場面はありますか？**
- **その時、何を使って、だいたい何分・何ステップかかりますか？**（音声入力／手入力／相手に渡す等）
- **相手を待たせることへの気まずさや、会話が止まる煩わしさはありますか？**
- **翻訳した内容を、後で見返したい・覚えたいと思うことはありますか？**
- **「その場の会話を成立させたい」と「後で覚えたい」は、同じ場面で起きますか？ それとも別ですか？**

#### B. 属性別に聞く（最大2問だけ選ぶ）

**駐在・出張層**

- 仕事中に翻訳・GPTを使う場面はどこですか？（会議／資料／チャット／現地スタッフへの指示 等）
- 機密・プライベートな内容が保存されることへの抵抗はどの程度ありますか？
- 仕事で使った表現は後で覚えたいですか？ それともその場限りで十分ですか？
- 中国語ができるようになると、仕事・生活で何が変わると思いますか？

**ポーカー層**

- 卓上やカジノ内で、実際にアプリを開けるタイミングはありますか？（会話中／離席中／ハンド後／食事中 等）
- 中国語→日本語の聞き取り補助は、場の空気やマナー的に使えそうですか？
- 卓上・フロア・レストランで「英語押し切り」では解決しない場面はどこですか？

**学習意欲低め／困っていない人**

- どの状況なら、中国語学習ではなくても中国語ツールを開きますか？
- 学習はしたくないが、会話補助だけなら使う可能性はありますか？
- 翻訳で困っていないなら、何があれば今のやり方から雇い替えますか？

#### C. 既に当方プロトタイプを触っている人だけ（時間が余れば）

- **このツールをどんな場面で開きましたか？**（会話中／会話後／思いついた時／空き時間）
- **何の代わり**として使いましたか？（GPT／Google翻訳／メモ／指差し）
- 使うのをやめた／忘れた場面はありますか？ その理由は？

#### D. 品質定義／避けたい体験（1問だけ）

- **いまのやり方で「これは避けたい・嫌だ」と感じている瞬間はありますか？**（例：翻訳待ちで相手を待たせる、自分に関係ない単語を覚える、翻訳機を相手に渡す、毎回人に聞く）
- 余裕があれば：**何があれば本当に解決した**と言える状態になりますか？

### CLOSE（0:27 – 0:30）— 引き換え・紹介

- もしこの課題が完全に解決したら、何が変わりますか？（**遂げたい進歩の言語化**）
- そのために **引き換えにしてもいい**ものは？（時間／お金／プライバシー／プライド／学習の苦労）
- 同じ悩みを持ちそうな知人を、**1人だけ**紹介いただけますか？
- 今日の話で「これだけは覚えてほしい」と思うことはありますか？

> ※ **「避けたい体験」は CHECK-D で聞き終えているはず**。CLOSE では繰り返さない。

## 🟢 GOOD SIGNALS（聞こえたら勝ち）

- 「**そう、それ！**」「**まさにそれ**」
- 「**最近もあって…**」（直近エピソードを自発的に語る）
- **数字**（◯円損した／◯回経験／◯時間悩んだ）
- **感情ワード**（悔しい／恥ずかしい／ナメられた／もったいない／疎外感）
- 既存解（翻訳・GPT・学習アプリ）への**具体的な**不満
- **「進歩」を自分の言葉で語る**（「本当は◯◯したかった」）
- 「**いくらなら払う**」を自発的に発言
- 「**人に紹介する**」を自発的に発言
- **現状で避けたい体験**（待たせる／関係ない単語を覚える／恥ずかしい等）が具体的に出る
- 前のめりになる、目が輝く

## 🟡 MIDDLE SIGNALS（要追加検証）

- 「たまにありますね」
- 「自分は大丈夫だが周りは」
- エピソードが**1年以上前**しか出ない
- 痛みはあるが「致命的というほどでは」
- 翻訳で「だいたい何とかなっている」

## 🔴 BAD SIGNALS（この人は対象外かも）

- 「困ってないですね」
- 「ChatGPTで十分です」「翻訳で完結してます」
- エピソードが**具体的に出ない**、抽象的
- 進歩の方向が **そもそも中国語ではない**（英語・別言語）
- 質問が**製品スペック**に向く（課題に共感していない）
- 興味が薄そう、反応が淡白

---

## ❌ DO NOT（絶対やらないこと）

- ❌ 製品を**最初に**見せない。聞かれても「最後の3分だけ」
- ❌ 「**使ってみたい？**」と聞かない（全員YESと答える嘘の検証）
- ❌ **値段を最初に**聞かない。最後に自然に
- ❌ **沈黙を埋めない**（5秒待つ）
- ❌ **仮説を守ろうと**しない（反証は宝物）
- ❌ **説得しない**（説明はOK、説得はNG）
- ❌ **誘導しない**（YES/NO質問は避ける）
- ❌ **「使ってますか？」だけで止めない**。**何分・何ステップ・誰と**まで聞く

## 🪄 沈黙・深掘り時の魔法の言葉

```
「もう少し具体的に…？」
「直近だといつでした？」
「その時、頭の中で何を考えてました？」
「他に似た場面ありますか？」
「これ以上ない、というほど痛かった経験は？」
「もし今その場面に戻れたら、何と言いたかったですか？」
「それで結局、どうなったんですか？」
「なぜそう感じたんでしょう？」
「本当は何をしたかったんですか？」
「いま、その状況で何を雇って間に合わせていますか？」
「そのとき、何が一番邪魔でしたか？」
「いまのやり方で『これは避けたい・嫌だな』と感じている瞬間はありますか？」
「やらずに済むなら、本当はやりたくないことはありますか？」
```

## ⚙️ Troubleshooting

| 状況 | 対応 |
|------|------|
| 「特に困ってない」と言われる | 「過去半年で『あれ？』『えっ？』と思った瞬間は？」と粒度を下げる |
| 話が脱線 | 「とても面白いお話ですね。一つだけ戻させてください。先ほどの◯◯について…」 |
| 沈黙が続く | 5秒待つ → 「考えるのに時間が必要ですよね。別の角度から聞きますね…」 |
| 製品について質問された | 「最後の5分でご紹介させてください。先に体験談をもう少しだけ…」 |
| 答えが抽象的すぎる | 「**具体例を1つ**教えていただけますか？」 |
| 「翻訳で何とかなる」で止まる | **手順・時間・相手の反応・自分の感情**まで具体化させる |
| 学習はしてないがマカオ等で中国語は触れる | CHECKは A を飛ばし B／C 中心。DEEPは卓・フロア・損に寄せる |
| 時間が押している | DEEPと CHECK-B（翻訳/GPT利用）を最優先、A は捨ててよい |
