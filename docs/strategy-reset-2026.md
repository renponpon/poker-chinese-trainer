# Phrabit 戦略リセット 2026

**作成日**: 2026-06-01  
**位置づけ**: 英語対応後の事業仮説を、アントレプレナーの教科書の顧客開発フレームに沿って再定義するメモ。  
**ステータス**: 仮説。市場規模・競合・価格・継続利用は未検証を含む。
**関連**: `docs/market-competitor-refresh-2026.md`, `docs/roadmap-2026-2028.md`

## 0. なぜリセットするか

Phrabit は当初「中国語圏ポーカー向けの中国語学習・会話補助アプリ」として始まった。

現在は、実装上も利用シーン上も次の形に近づいている。

> 海外の現場で使う短文を翻訳し、必要なものだけ保存し、あとで瞬間作文として復習するアプリ。

中国語だけでなく英語にも対応したことで、事業仮説の中心は「中国語」ではなく、次の問いに移る。

> 翻訳アプリでその場をしのいだ言葉を、次に自分で使える状態にすることに、課金価値はあるか。

ただし、最初から広い語学学習市場を狙うと競合が強すぎる。
そのため、検証の入口は引き続き「海外ライブポーカーをする日本人」でよい。
理由は、ユーザーに近く、現場で観察でき、言葉の不足が体験・関係性・金銭に影響しやすいから。

## 1. 顧客開発上の現在地

アントレプレナーの教科書的には、現時点はまだ「会社を大きくする段階」ではない。

今やるべきことは、顧客発見のやり直しに近い。

| フェーズ | Phrabit の現在地 |
|---|---|
| 仮説構築 | 中国語限定から、英中対応・多言語前提へ再定義が必要 |
| 顧客発見 | 中国語圏インタビューは実施済み。英語/WSOP文脈は不足 |
| 顧客実証 | まだ弱い。実際に使い続けるか、課金するかは未検証 |
| 顧客開拓 | まだ早い。告知は学習と検証目的で小さく始める |

したがって、直近の目的は売上最大化そのものではなく、売上最大化につながる勝ち筋を特定すること。

## 2. 製品仮説

### 2-1. 旧仮説

> マカオでポーカーをする日本人は、中国語で言いたい一言が出ないため、AIで自然な中国語を作り、ドリルで覚えたい。

この仮説は完全には捨てない。
ただし、今は狭すぎる。

### 2-2. 新仮説

> 海外の現場で同じような会話・手続き・注文・確認を繰り返す日本人は、翻訳アプリでその場をしのぐだけでは不十分で、使った表現を保存し、次回すぐ言えるように復習したい。

### 2-3. 一文での製品定義

> Phrabit は、海外現場の「今これ何て言う？」を、次に自分で言えるフレーズに変えるアプリ。

### 2-4. 提供価値

| 価値 | 内容 |
|---|---|
| その場で通じる | 日本語から相手の言語へ素早く翻訳する |
| 相手を理解する | 相手の言語から日本語へ戻せる |
| 残せる | 必要なフレーズだけライブラリに残す |
| 覚えられる | 保存したフレーズを瞬間作文で復習する |
| 自分用になる | 自分が実際に使った表現が蓄積される |

## 3. 顧客仮説

### 3-1. 第1ターゲット

> 海外ライブポーカーをする日本人。

特に、次の人。

- 海外カジノやポーカールームに複数日滞在する
- 英語または中国語が完璧ではない
- 卓上、フロア、レストラン、移動で同じような一言に何度も困る
- 翻訳アプリやChatGPTで調べるが、次回もまた調べている
- 語学を少しでも改善したい気持ちがある

### 3-2. 第1ターゲットの中で強い層

「ポーカーをする人」全員ではない。
強いのは次の層。

| 層 | 期待値 |
|---|---|
| 現地で数日以上プレイする人 | 同じ場面が繰り返されるため保存・復習価値が出やすい |
| 英語/中国語で少し悔しい経験がある人 | ドリル価値が出やすい |
| 翻訳アプリをすでに使っている人 | 代替手段からの乗り換え検証がしやすい |
| 自分のフレーズを増やしたい人 | ライブラリ・追加フレーズ生成が刺さりやすい |

### 3-3. 弱い層

| 層 | 理由 |
|---|---|
| 翻訳アプリで十分な人 | 復習・保存に価値を感じにくい |
| 語学改善意欲がない人 | ドリルが重く感じる |
| ほぼ英語だけで困らない人 | 現場課題が弱い |
| 旅行が単発の人 | 継続利用しにくい |

## 4. 課題仮説

ユーザーが本当に困っているのは「翻訳できないこと」だけではない。

より深い課題は次の通り。

| 課題 | 説明 |
|---|---|
| 体験品質の低下 | その場で言葉が出ず、会話や手続きが止まる |
| 機会損失 | 卓上会話、交渉、注文、確認でチャンスを逃す |
| 関係性損失 | 輪に入れない、相手との距離が縮まらない |
| 自尊心損失 | また同じ表現を調べている、という悔しさが残る |
| 学習断絶 | 調べた表現が次回使える知識にならない |

Phrabit が解くべき中心課題はこれ。

> 翻訳した瞬間と、次に自分で使える状態の間にある断絶。

## 5. 既存代替手段・競合仮説

### 5-1. 直接競合ではなく、代替手段が強い

Phrabit の最大競合は、同じ機能を持つアプリではない。
ユーザーが今すでに使っている代替行動が競合になる。

| 代替手段 | 強み | Phrabit が勝てる可能性 |
|---|---|---|
| Google翻訳 | 速い、無料、多言語、会話・カメラが強い | 保存・復習・自分用化では弱い |
| DeepL | 自然な翻訳、信頼感 | 現場会話・ドリル化では弱い |
| ChatGPT / Gemini | 文脈説明、言い換え、自由度 | 毎回プロンプトが必要。保存・反復が面倒 |
| Duolingo / HelloChinese | 継続設計、教材量、ブランド | 自分の現場フレーズではない |
| Anki / Quizlet | SRS、自由度 | 作成負荷が高い |
| メモアプリ | すぐ残せる | 翻訳・音声・復習につながらない |
| 人に聞く | 文脈に合いやすい | いつでも聞けない。蓄積されない |

### 5-2. 差別化仮説

Phrabit は「翻訳精度」だけで勝たない。

勝ち筋は次の組み合わせ。

> 現場翻訳 + 自分用ライブラリ + 瞬間作文ドリル + 言語別の蓄積。

特に重要なのは、ユーザーが「自分が実際に使いそう」と思えること。
固定教材ではなく、自分の現場から生まれたフレーズであることが差別化になる。

## 6. 市場仮説

### 6-1. 市場の見方

市場は1つではなく、広げ方で分ける。

| レイヤー | 内容 | Phrabit の位置づけ |
|---|---|---|
| Beachhead | 海外ポーカー日本人 | 最初の検証市場。狭いが観察しやすい |
| Adjacent | 海外旅行・長期滞在・出張の日本人 | 拡張市場。英語対応で近づいた |
| Larger Market | 語学学習者全般 | 大きいが競合が強い。初期は狙いすぎない |

### 6-2. 外部データメモ

- JNTO発表ベースの記事では、2025年の日本人海外旅行者数は約1,473万人。2019年比ではまだ約27%少ないが、前年比では増加している。  
  参考: https://www.travelvoice.jp/english/japanese-international-travelers-were-still-down-27-to-14-7-million-in-2025-over-2019
- JTB Outbound Tourism Report 2025 は、日本人海外旅行市場の回復、座席供給、若年層の動きなどを扱っている。  
  参考: https://www.tourism.jp/en/tourism-database/publications/jtb-outbound-report-2025/
- 2025 WSOP Main Event は9,735エントリーで、過去3番目規模。海外ポーカー自体は大きなイベント性を持つ。  
  参考: https://www.wsop.com/news/final-nine-players-set-for-the-2025-world-series-of-poker-main-event-final-table/
- Google翻訳は2024年に110言語を追加。翻訳だけで勝つのは難しい。  
  参考: https://support.google.com/translate/answer/15139004?hl=en
- DeepL は多数の言語組み合わせに対応し、日英中も含む。  
  参考: https://support.deepl.com/hc/en-us/articles/360019925219-DeepL-Translator-languages
- Duolingo は2025年Q1に有料会員1,000万人超を発表。語学学習市場の大手は強い。  
  参考: https://investors.duolingo.com/node/10916/pdf

### 6-3. 市場規模の仮説

現時点では、海外ポーカーだけで大きな利益を最大化するのは難しい可能性が高い。

一方で、海外ポーカーは次の意味で良い。

- ユーザーに直接会える
- 利用場面が具体的
- 英語・中国語の両方が自然に発生する
- 使わない理由まで観察できる
- 口コミの初速を作りやすい

つまり、海外ポーカーは「最大市場」ではなく「最初に勝ち筋を見つける市場」と位置づける。

## 7. 収益仮説

### 7-1. 初期の考え方

今すぐログイン必須・課金必須にすると、検証母数が減る。
したがって、当面はゲスト利用を残し、価値を感じた後にログイン・課金へ進む形がよい。

### 7-2. 課金候補

| 方式 | 初期適性 | コメント |
|---|---|---|
| 広告 | 低 | 現場利用の邪魔になる。単価も低い可能性が高い |
| 月額 | 中 | 継続利用が見えたら本命 |
| 買い切り | 中 | ポーカー遠征・旅行前の心理と相性はある |
| フリーミアム | 高 | 無料で翻訳、保存/ドリル/生成上限/同期を有料化しやすい |
| AI教師・パーソナライズ | 中〜高 | 自己紹介や目的別フレーズ生成と相性がよい |
| フレーズパック販売 | 中 | ポーカー、旅行、出張などテーマ別に可能 |
| 法人/チーム | 低〜中 | 将来。今は重い |

### 7-3. 暫定方針

短期は課金導線を急がず、次を確認する。

1. 保存されるか
2. ドリルに追加されるか
3. 翌日以降に戻ってくるか
4. 自分用フレーズを増やしたいか
5. 有料なら何に払うと言うか

## 8. 重要KPI仮説

インストール数やアクセス数より、次を見る。

| 段階 | 見る指標 |
|---|---|
| 初回価値 | 初回翻訳完了率、翻訳速度への不満 |
| 保存価値 | ライブラリ保存数、ドリル追加率 |
| 学習価値 | 初回ドリル実行率、翌日復習率 |
| 継続価値 | D1/D7再訪、同じ言語での再利用 |
| 多言語価値 | 中国語/英語の切替回数、言語別保存数 |
| 課金価値 | 有料でも欲しい機能、支払い意思、価格反応 |

特に重要なのは、次の3つ。

1. 翻訳後に保存/ドリル追加されるか
2. 翌日以降に戻ってくるか
3. Google翻訳やChatGPTではなくPhrabitを開く理由があるか

## 9. 拡張仮説

### 9-1. 英語対応の意味

英語対応は、単なる言語追加ではない。
Phrabit の本質が「中国語アプリ」なのか「現場フレーズ学習アプリ」なのかを検証するための重要な実験になる。

### 9-2. 拡張順序

| 順序 | 対象 | 判断基準 |
|---|---|---|
| 1 | 中国語・英語の海外ポーカー | 直近の現場検証。マカオ/WSOP文脈 |
| 2 | 海外旅行・長期滞在 | ポーカー外でも保存/復習価値が出るか |
| 3 | 出張・駐在 | 支払い意思が高いか。業務品質要件に耐えるか |
| 4 | 多言語一般 | 勝ち筋が繰り返せると分かった後 |

### 9-3. 拡張してよい条件

次が揃うまでは、広げすぎない。

- 特定セグメントで繰り返し使われている
- 保存/ドリル追加が自然に起きている
- 代替手段では満たせない理由が言語化できている
- 課金候補機能が見えている
- 言語を増やしてもUXが複雑になりすぎない

## 10. 現時点の最重要リスク

| リスク | 内容 | 検証方法 |
|---|---|---|
| 翻訳だけで十分 | ユーザーが保存・復習を使わない | 利用ログ、実地観察、翌日再訪 |
| ドリルが重い | 現場ユーザーは学習したくない | ドリル追加率、復習率、インタビュー |
| 英語はGoogle翻訳で十分 | 英語ではPhrabit差別化が弱い | 英語利用者の保存/復習行動を見る |
| 市場が狭い | ポーカーだけでは伸びない | 旅行/出張への隣接検証 |
| 速度不満 | 現場では待てない | 実機テスト、速度モード利用率 |
| ログイン障壁 | データ把握したいがログイン必須化で離脱 | ゲスト利用後のログイン誘導を検証 |

## 11. 次に検証する問い

仮説再定義後、次に見るべき問いはこれ。

### プロダクト利用

1. 初見で「何のアプリか」分かるか
2. Google翻訳ではなくPhrabitを開く理由があるか
3. 翻訳速度は現場で許容されるか
4. 保存・ドリル追加は自然か
5. 英語と中国語の切替は混乱しないか

### 顧客課題

1. ユーザーは本当に同じ場面に何度も遭遇しているか
2. その表現を次回言えるようになりたいか
3. どの場面なら「覚える価値」があるか
4. どの場面は翻訳だけで十分か

### 課金

1. 何に払うなら納得するか
2. 月額か、遠征前の買い切りか
3. 無料枠の上限はどこなら自然か
4. ログイン・同期・自分用フレーズ生成に価値があるか

## 12. 検証優先度

全部を同時に検証しない。
事業判断に影響が大きく、間違っていたら方向転換が必要なものから潰す。

| 優先度 | 仮説 | 現在の確信度 | 外れていた時の影響 | 次の確認方法 |
|---|---|---|---|---|
| P0 | 翻訳後に保存・ドリル化したい人がいる | 中 | 差別化の中心が崩れる | 保存率、ドリル追加率、実地観察 |
| P0 | 海外ポーカーは最初の検証市場として機能する | 中 | 初期獲得チャネルを変える必要 | X告知、知人配布、現地テスト |
| P0 | 英語でも「保存・復習」価値は成立する | 低〜中 | 英語対応の優先度を下げる | WSOP/英語利用者の行動を見る |
| P1 | 翻訳速度は現場利用に耐える | 中 | 利用前に離脱する | 実機テスト、速度不満の記録 |
| P1 | ゲスト利用を残した方が初期検証に有利 | 中 | データ取得が弱くなる | ゲスト利用後のログイン率を見る |
| P1 | 課金価値は翻訳回数ではなく保存・復習・自分用化にある | 中 | 料金設計を変える必要 | 価格質問、仮課金導線、手動ヒアリング |
| P2 | 旅行・出張にも横展開できる | 低 | 市場拡張計画を見直す | ポーカー外の小規模ヒアリング |

現時点で一番大事なのは、Phrabit が「翻訳アプリの劣化版」ではなく、「調べた言葉が自分の資産になる道具」として使われるかどうか。

## 14. 現時点の暫定結論

Phrabit は「中国語学習アプリ」としてではなく、次のカテゴリで捉える方がよい。

> 現場フレーズを翻訳・保存・復習する、個人用フレーズ運用ツール。

最初の市場は海外ポーカーでよい。
ただし、事業利益の最大化を考えるなら、ポーカーだけで完結させない。

勝ち筋は、海外ポーカーで高密度に検証し、そこで見えた「翻訳から復習への橋渡し」が旅行・長期滞在・出張にも再現するかを確かめること。
