# ご意見番候補リスト（ドラフト）

中国語**学習**に強い発信者を、X / note / YouTube から抽出した**サーチ補助用**リスト。  
**未コンタクト・未許諾**のため、DMする際は必ずプロフィールと直近投稿を読み、文脈に合わせて文面を組み立てること。

更新方針: インタビューで「紹介してほしい専門家」を聞き、**リストを差し替え・追記**していく。

---

## 使い方（最短）

1. **目的**: 売り込みではなく「中国語学習サービスの現場感・離脱要因」を聞く。
2. **優先**: 下記「★学習コア」から5〜8名に短いDM。返信なしなら次のバッチへ。
3. **避ける**: 政治ニュース主導アカウント（学習インタビューのノイズになりやすい）。

---

## カテゴリ凡例

| タグ | 意味 |
|------|------|
| ★学習コア | 中国語の**学び方・教材・試験**が主題に近い |
| ビジネス中国 | 経済・企業・出張文脈が強い |
| 講師・教室 | サービス提供者（協力・紹介の可能性と利益相反の両方に注意） |
| 体験記 | 独学・会社員ベースのリアル語り |
| 参考外 | 今回の「学習ご意見番」には不向きになりやすい |

## DM時の一言テンプレ（コピペ用・必ず自分語に直す）

**学習系note作者向け（短文）**

> はじめまして。中国語学習アプリを試作していて、**「続かない理由」と「現場で言えない理由」が本当に同じかどうか**をプロの視点で伺いたいです。15分だけオンラインでもよいでしょうか。売り込みはせず、録音可なら学習メモとして使わせてください。

**翻訳者・ビジネス文脈向け**

> はじめまして。日本語→中国語の**自然さ**を重視した学習ツールを作っています。AI生成文の「現場感」について、プロの感覚で**10分だけ**忌憚なく教えていただけると助かります。
