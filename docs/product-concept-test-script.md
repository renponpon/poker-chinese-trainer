# 製品コンセプト検証スクリプト

**作成日**: 2026-05-19  
**対象**: V0.2 Concept Validation Build  
**目的**: 第三フェーズ「製品コンセプトの検証と洗練」で、同じ説明・同じ質問で顧客反応を比較する。  
**参照元**: `product-concept-validation-plan.md`、`product-iteration-priority.md`、`mvp-next-iteration-spec.md`

---

## 0. この検証で知りたいこと

今回の目的は、製品を褒めてもらうことではない。  
知りたいのは、次の5点。

1. Google翻訳/ChatGPTではなく、このアプリを開く理由があるか
2. `中→日` が入ることで、会話中に使う理由が増えるか
3. 全件ライブラリに残ることが価値になるか
4. 「ドリルにも追加する」を選べることが自然か
5. 有料でも使う可能性があるか

---

## 1. 対象者

### 優先対象

- マカオ/中国語圏でポーカーをする日本人
- 中国語で困った経験があり、少しでも改善したい人
- 既にプロトを触った人
- 直近で中国語を使う予定がある人

### 補助対象

- 駐在員/駐在帯同家族
- 中国語を学習中で、実生活との接続に困っている人
- 実務中国語でニュアンスに困った人

### 今回は優先しない対象

- 中国語に困っていない人
- 翻訳/笑顔で十分な人
- 中国語ができるようになりたい気持ちがない人

## 2. 冒頭説明

そのまま読める説明。

```text
今日は、完成品の評価ではなく「方向性が合っているか」を見たいです。

今までは、日本語を中国語にしてドリルにするアプリでした。
ただ、インタビューを重ねると、実際には「その場で相手の中国語を理解したい」「翻訳した内容をあとで見返したい」「覚えたいものだけ練習したい」という声が多く出ました。

そこで今回、Google翻訳みたいにその場で使えるけど、調べた表現がライブラリに残って、覚えたいものだけドリルに入れられる形に変えています。

売り込みではないので、使わなさそう・面倒そう・Google翻訳で十分、と思ったらそのまま教えてください。
```

## 3. 画面を見せる順番

### 画面1: ホーム

見せるもの:

- `今すぐ追加`
- `今日のドリル`
- `ライブラリ`
- 右上ログインボタン
- ログインなしで使える説明

確認すること:

```text
最初に開いた時、「何ができるアプリか」は分かりますか？
ログインなしで使えることは安心ですか？
右上ログインは自然ですか？
```

見るシグナル:

- 迷わず `今すぐ追加` を押せるか
- ログインを求められないことを良いと感じるか
- 自分用コードが消えたことに違和感がないか

---

### 画面2: 追加/翻訳

見せるもの:

- `日本語 → 中国語`
- `中国語 → 日本語`
- 入力欄
- 音声入力
- `翻訳`
- `ドリルにも追加する`

確認すること:

```text
この「日本語→中国語」と「中国語→日本語」の切り替えは、使う場面がイメージできますか？
```

```text
中国語→日本語は、どんな場面で使いそうですか？
卓上、レストラン、フロア、タクシー、仕事などで思い当たる場面はありますか？
```

```text
「ドリルにも追加する」は、最初からオンの方がいいですか？
それとも、毎回自分で選びたいですか？
```

見るシグナル:

- 具体的な使用場面が出る
- `中→日` に反応する
- ドリル追加のオン/オフに自分なりの意見が出る
- 「翻訳ボタンだけならGoogle翻訳でいい」と言うか

---

### 画面3: 翻訳結果

見せるもの:

- 日本語
- 中国語
- ピンイン
- 解説
- 音声再生
- ドリルへ導線

確認すること:

```text
この結果の中で、現場で一番見たいのはどこですか？
中国語、ピンイン、音声、解説、想定返答、類似フレーズのどれが重要ですか？
```

```text
この解説量は多いですか、ちょうどいいですか？
会話中に見るなら、もっと短い方がいいですか？
```

見るシグナル:

- 音声/ピンインへの反応
- 解説が現場では重いと言われるか
- 「あとで見るなら良い」と言うか
- 類似フレーズ/想定返答への追加ニーズが出るか

---

### 画面4: ライブラリ

見せるもの:

- 全件一覧
- 検索
- 状況カテゴリフィルタ
- ドリル対象/ライブラリのみフィルタ
- 日→中 / 中→日 バッジ
- ドリルに追加/外す

確認すること:

```text
翻訳したものが全部ここに残るとしたら、見返しそうですか？
```

```text
見返す時は、検索で探しそうですか？
それとも、ポーカー卓・レストラン・移動みたいな状況別で探しそうですか？
```

```text
「ライブラリには残すけど、ドリルには入れない」という分け方は自然ですか？
```

見るシグナル:

- 全件保存に価値を感じる
- 状況別カテゴリの具体案が出る
- ライブラリのみ/ドリル対象の分離に納得する
- 逆に「面倒」「全部ドリルでいい」「全部保存は嫌」が出る

---

### 画面5: ドリル

見せるもの:

- ドリル対象だけ出る
- Bad / Good / Perfect
- 中→日フレーズも出題できる

確認すること:

```text
このドリルは、いつ使いそうですか？
現地に行く前、カジノに向かう前、寝る前、暇な時など、具体的なタイミングはありますか？
```

```text
ドリルに入れる数が増えすぎると嫌になりそうですか？
どれくらいなら続きそうですか？
```

見るシグナル:

- 具体的な利用タイミングが出る
- 1回あたりの許容量が出る
- 通知/予定化の必要性が出る
- 「ドリルは使わないがライブラリは使う」が出る

## 4. 最後に聞く質問

### 4-1. 既存解との比較

```text
このアプリを見た後でも、Google翻訳やChatGPTで十分だと思いますか？
それとも、このアプリを使う理由がありそうですか？
```

深掘り:

```text
使う理由があるとしたら、どの機能ですか？
使わないとしたら、何が足りないですか？
```

### 4-2. 使用場面

```text
次に中国語で困った時、このアプリを開きそうな場面はありますか？
あるとしたら、どこで誰と何をしている時ですか？
```

### 4-3. 保存とプライバシー

```text
翻訳した内容がこの端末やアカウントに残ることに抵抗はありますか？
保存したくない内容はありますか？
```

### 4-4. 価格感

```text
無料なら使うと思いますか？
もし有料だとしたら、月いくらくらいまでなら検討しますか？
月300円、500円、1000円、1500円だと、それぞれどう感じますか？
```

ポーカー層向け追加:

```text
マカオ滞在中だけ使うなら、月額より買い切りや1ヶ月パスの方が自然ですか？
```

ビジネス/駐在層向け追加:

```text
仕事で使う場合、個人で払う感覚ですか？
それとも会社経費にできる可能性がありますか？
```

### 4-5. 紹介

```text
同じように中国語で困っていて、この方向性に反応しそうな人は1人いますか？
```

## 6. 成功/失敗判定

### 強い成功

以下のうち3つ以上。

- 「Google翻訳よりこっちを使う場面がある」
- 具体的な使用場面が即出る
- `中→日` への反応が強い
- 全件ライブラリ/状況別整理に強く反応
- 「今使いたい」「試したい」
- 価格に具体反応がある
- 紹介候補が出る

### 中間

- 便利そうだが、Google翻訳で足りる
- ライブラリは良いがドリルは弱い
- ドリルは良いが会話中には開かない
- 無料なら使うが有料は弱い

### 失敗

- 使う場面が浮かばない
- Google翻訳/ChatGPTで十分
- 保存が嫌
- ドリル化が重い
- 何をするアプリか伝わらない

## 7. 次の意思決定

5〜8人に見せた後、次を決める。

1. `中→日` をコアに昇格するか
2. ライブラリ/カテゴリを主要価値として前面に出すか
3. ドリルを主役にするか、副産物にするか
4. ポーカー特化で磨くか、生活者/駐在にも広げるか
5. 課金導線を試すか、まだ無料検証を続けるか
