# ペルソナ1: れん

**中国語圏ライブポーカーで勝負する日本人ポーカープレイヤー**

検証ステータス: **🟢 検証済**（プロダクトオーナー自身がN=1として該当）

## 基本プロフィール

| 項目 | 内容 |
|---|---|
| 名前 | れん |
| 年齢 | 31歳 |
| 出身 | 大阪府 |
| 居住地 | 日本とマカオ・台北・マニラを行き来。拠点は大阪 |
| 職業 | プロ寄りのポーカープレイヤー。オンライン、ライブ、たまにコーチング |
| 年収感 | 年によって大きく変動。勝ち年は1,000万円超、悪い年は生活費程度 |
| 中国語レベル | HSK 3〜4級相当。単語は拾えるが、卓上の自然会話はかなり厳しい |
| 英語 | 旅行・ポーカー用語は対応可能 |
| 趣味 | ポーカー、MMA観戦、サウナ、海外グルメ、YouTube、投資系情報収集 |
| 生活 | 不規則。夜型。遠征中はカジノ、ホテル、ジム、カフェの往復 |
| 旅行のこだわり | 安すぎるホテルは避ける。カジノアクセス、睡眠環境、食事、Wi-Fi重視 |
| IT/スマホリテラシー | 高い。スマホ完結が好き。Notion、Google翻訳、DeepL、ChatGPT、ポーカー管理アプリを使う |

## 情報収集

| 項目 | 内容 |
|---|---|
| 主なメディア | X、YouTube、Discord、LINEオープンチャット、ポーカーコミュニティ、海外カジノ情報ブログ |
| 見ている内容 | トーナメント情報、キャッシュゲーム事情、各国のポーカールーム情報、強いプレイヤーの発信 |
| 最近気になる話題 | マカオのポーカールームの変化、中国人富裕層プレイヤー動向、海外キャッシュの期待値、AIでの語学学習、海外移住・税金 |

## 性格

合理的で、基本的に「勝てるかどうか」で物事を見る。  
ただし完全な機械的人間ではなく、テーブルの空気、相手との距離感、常連ネットワークの価値も理解している。

表向きは「中国語はなくてもプレイできる」と言うが、実際には中国語ができるプレイヤーを見ると焦る。負けず嫌いで、情報弱者になることを嫌う。

## 日常・仕事

れんの仕事は単にカードをプレイすることではない。  
良いゲームを探し、テーブルを選び、相手の性格や癖を読み、相手にどう見られるかを管理し、期待値のある環境に長く居続けることが仕事。

ライブポーカー中の頻出場面:

- 中国人プレイヤー同士がハンドについて話している
- ディーラーやフロアとのルール確認が発生する
- 相手が冗談や煽りを言ってくる
- オールイン後やショーダウン後に会話が起きる
- 常連同士の会話に入れず、情報を逃す
- トラブル時に自分の主張を十分に説明できない

## 課題: 不満・不便・不安

### 不満
一般的な中国語教材をやっても、カジノで使う言葉に全然つながらない。  
「私は学生です」「今日は天気がいいです」ではなく、「今のアクションはまだ完了してない」「彼が先にショーした」「このポット計算おかしくない？」が言いたい。

### 不便
DeepLや翻訳アプリは使えるが、テーブル中はテンポが命。スマホを出して翻訳している時点で、会話の流れから外れる。  
翻訳した文章をその後に反復練習する仕組みがない。

### 不安
聞き取れていないせいで、自分が損しているかもしれない。  
ルール上の不利、相手の癖、場の空気、プレイヤー間の人間関係など、見えない情報を取り逃している感覚がある。

## 達成したいこと

「中国語が上手くなること」ではない。

- テーブルで自然に会話に入れる
- 中国人プレイヤーの雑談から情報を拾える
- ハプニング時に自分の主張を通せる
- 冗談や煽りに軽く返せる
- 「こいつ分かってないな」と思われない
- 現地コミュニティに入り、良いゲーム情報を得られる
- 中国語が理由でEVを落としたくない

## 見えないインサイト

深い本音: **「ポーカーでは強い自分が、中国語になると急に弱者になるのが嫌」**

本人は「EVのため」「情報収集のため」と合理的に説明するが、心の奥には:

- 中国人プレイヤー同士が笑っている時、自分が笑われている気がする
- 会話に入れないことで、テーブル上の存在感が薄くなるのが悔しい
- 本当は"海外でも堂々と立ち回れるプレイヤー"として見られたい
- 中国語ができる日本人プレイヤーに対して、少し劣等感がある
- 「カードでは勝てるのに、言葉で下に見られる」のが許せない

## 刺さる訴求

「中国語を勉強しよう」では弱い。れんに刺さるのは:

> マカオのテーブルで、聞き取れずに流される側から、会話を拾って返せる側へ。

> ポーカー中に本当に使う中国語だけを、反射で出るまで鍛える。

## 痛みの定量（自己申告）

- 頻度: 数日に1回
- 通常時の痛みスコア: 5/10
- ハプニング発生時: 10/10
- 解決対価: 完全解決なら月¥50,000、サブスクなら月¥数千

# ペルソナ2: たけし

**中国系企業で働く日本人ビジネスパーソン**

検証ステータス: **🔴 未検証**（仮説のみ、Phase 2 で検証対象）

## 基本プロフィール

| 項目 | 内容 |
|---|---|
| 名前 | たけし |
| 年齢 | 38歳 |
| 出身 | 神奈川県 |
| 居住地 | 東京都江東区 |
| 職業 | 中国系IT/メーカー/商社の日本法人勤務。事業開発または営業企画マネージャー |
| 年収感 | 750万〜1,000万円 |
| 中国語レベル | HSK 4級前後。読み書きは多少できるが、会議の即答が苦手 |
| 英語 | ビジネス初中級 |
| 家族 | 既婚。子ども1人 |
| 趣味 | ガジェット、ゴルフ、ランニング、ビジネス書、Netflix、出張グルメ |
| 生活 | 平日は会議が多い。夜に少し勉強するが、疲れて続かない |
| 旅行のこだわり | 出張では効率重視。空港アクセス、ホテルの清潔さ、ラウンジ、移動ストレス |
| IT/スマホリテラシー | 高い。Slack/Teams/WeChat/Notion/ChatGPTを使う。語学アプリも複数試した経験あり |

## 情報収集

| 項目 | 内容 |
|---|---|
| 主なメディア | NewsPicks、日経電子版、X、LinkedIn、YouTube、Voicy、社内WeChat、業界ニュース |
| 見ている内容 | 中国テック企業、EV、半導体、AI、日中ビジネス、マネジメント、転職市場 |
| 最近気になる話題 | 中国企業のグローバル展開、日本市場での中国ブランドの伸び、AI翻訳の進化、自分の市場価値 |

## 性格

真面目で責任感が強い。社内では調整役として評価されている。  
一方で、内心では「自分は日本市場担当として必要とされているだけで、中国本社の本流には入れていないのでは」と感じている。

プライドは高いが、それを表に出すタイプではない。  
会議で聞き取れなかった時も「すみません、もう一度お願いします」と言えるが、何度も聞くのは避ける。結果として曖昧な理解のまま進めてしまう。

## 日常・仕事

中国本社と日本法人の間に立つ仕事。主な業務:

- 中国本社の方針を日本市場向けに翻訳・調整する
- 日本の顧客要望を中国側に伝える
- 定例会議で進捗報告する
- 中国人上司や本社メンバーとWeChatでやり取りする
- 日本側の事情を説明し、納期や仕様の調整を行う
- 中国側の決定を日本チームに伝える

一見すると「中国語が少しできる日本人」として重宝されているが、実際には**重要な意思決定ほど中国語で速く進み、たけしは後から理解する**ことが多い。

## 課題: 不満・不便・不安

### 不満
中国語学習アプリで勉強しても、実際の会議で使う言葉にならない。  
必要なのは「レストランで注文する表現」ではなく、「その前提は日本市場では成立しません」「このKPIは現場実態とズレています」と言える力。

### 不便
会議後にChatGPTや翻訳ツールで表現を調べるが、その場で言えなかった言葉は後から分かっても遅い。  
調べた表現を反射的に使えるようにする練習導線がない。

### 不安
中国語会議で発言量が少ないことで、上司から「日本市場担当としては便利だが、幹部候補ではない」と見られているのではないかと不安。  
実績や判断力が、言語能力のせいで過小評価されていると感じている。

## 達成したいこと

「中国語ペラペラ」ではない。

- 中国語会議で置いていかれない
- 必要な時にその場で反論・補足できる
- 中国人上司に対して、根拠を持って主張できる
- 自分の成果を中国語で正しくアピールできる
- 日本側の事情を"言い訳"ではなく"戦略的判断"として伝えたい
- 本社メンバーから「彼は分かっている」と認められたい
- キャリア上、中国語を弱点ではなく武器にしたい

## 見えないインサイト

深い本音: **「自分の実力が、中国語のせいで一段低く見られているのが悔しい」**

表向きは「業務効率のため」「会議をスムーズにするため」と言うが、心の奥では:

- 本当は中国人上司にもっと評価されたい
- 日本人社員の中で「中国語ができる側」に立ちたい
- 会議で黙っている自分を、能力が低い人間のように見られたくない
- 将来の昇進や転職で、中国語を強みにしたい
- 「日本市場担当の便利屋」で終わりたくない
- 中国本社の中枢に近い人材として扱われたい

## 刺さる訴求

> 中国語会議で、聞くだけの人から、主張できる人へ。

> あなたが会議で言えなかった一言を、次回その場で言える武器に変える。

## たけしを検証するインタビュー質問

Phase 2 のインタビューで以下を確認する:

1. 「会議で言えなかった一言」を**直近で具体的に**挙げられるか
2. 「中国語のせいで評価が低い」と感じるエピソードはあるか
3. 現状の対処（翻訳アプリ・あとで調べる）に**不満を持っている**か
4. 「自信」「評価」「キャリア」という感情ワードが**自発的に**出るか
5. 月いくらまで払う意思があるか

# アーリーアダプター仮説

Phase 2 で**検証対象**となる、アーリーアダプター（イノベーター）の特徴仮説。

## 仮説5項目

| # | 特徴 | 検証方法 |
|---|---|---|
| 1 | 中国語学習を**自己流で組んでいる**（ChatGPT/Anki/独自カードを既に使用） | インタビューで「現在の学習方法」を聞く |
| 2 | 新しいアプリを**月3つ以上試している** | インタビューで「最近試したアプリ」を聞く |
| 3 | 直近半年で**困った具体エピソードを2つ以上**持つ | 自然な会話で出るかを観察 |
| 4 | 自分の課題を**自分で言語化できる**（抽象論で終わらない） | 対話の中で観察 |
| 5 | SNSで**何かを発信している**（伝道力の指標） | 事前リサーチ（Xなど） |

→ 5項目中**3つ以上**当てはまる人をアーリーアダプターと暫定定義。

## 想定されるアーリーアダプター像

- ChatGPTを毎日使っていて、語学学習にも使っている
- DuolingoやAnkiを試したが「ちょっと違う」と感じて自分なりに改造している
- 新しいプロダクトを「ベータ版」段階で試すのが好き
- 自分の悩みを構造的に語れる
- 周囲に学習法を発信したり、勧めたりしている
