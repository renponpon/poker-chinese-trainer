# 競合の全体構造

中国語学習関連サービスを**5つのカテゴリ**に分類する。

| カテゴリ | 代表例 | 顧客が"奪われる"理由 |
|---|---|---|
| A. 大手語学アプリ | Duolingo / HelloChinese / SuperChinese | 続きやすさ・認知度 |
| B. AI翻訳・調べる手段 | ChatGPT / Gemini / DeepL / Google翻訳 | 即時性・無料 |
| C. AI会話トレーナー | MandaLearn / SureChinese / Talkpal | AIとの対話練習 |
| D. SRS・フラッシュカード系 | Anki / Quizlet / Mandarin Forge | 反復定着の科学 |
| E. パーソナルフレーズ集系 | Kippy / QuickPhrase / リアル中国語会話 | 自分用フレーズ |

→ 我々（SayItNext）は **B × D × E のクロスポジション**にいる。これは現時点で**直接的な競合がいない**ことを意味する一方、**カテゴリの認知度が低い**というリスクも意味する。

# A. 大手語学アプリ

## A-1. Duolingo（中国語コース）

### 概要

| 項目 | 内容 |
|---|---|
| 運営 | Duolingo, Inc.（米国・上場企業） |
| 中国語コース対象 | 初心者〜中級下 |
| 料金 | 無料（Super Duolingo: 月¥1,150） |
| グローバルユーザー | 月間アクティブユーザー1億人超 |
| 日本での認知度 | 極めて高い |

### 強み

- ゲーミフィケーション（ストリーク、ランキング、ハート）で**続けやすい**
- 基礎単語・四声・基本文法を**体系的**に学べる
- ネイティブ音声による発音判定
- 無料で主要機能が使える
- 認知度・ダウンロード数で他を圧倒

### 弱み

| # | 弱点 | 出典 |
|---|---|---|
| 1 | スピーキング力が伸びにくい（読み書き中心） | 複数レビュー[^1] |
| 2 | 中級以上には物足りない | 複数レビュー |
| 3 | ピンインと漢字がセットで覚えられない | レビュー |
| 4 | 文法説明が限定的 | レビュー |
| 5 | 試験対策に不向き | レビュー |
| 6 | 台湾華語に非対応 | レビュー |
| 7 | **現場で使う言葉に繋がらない**（教科書中国語） | れんの実体験 |
| 8 | **個人化されていない**（全員同じカリキュラム） | 構造的欠陥 |

### 当方との位置関係

- **間接競合**だが「最大の敵」とも言える
- 顧客の「学習時間」を奪う最大の存在
- 多くの人が**「Duolingoで続かなかった経験」**を持つ → 我々の差別化メッセージの基盤
- ChatGPTより認知度で勝るため、**ChatGPT vs SayItNext** よりも **Duolingo vs SayItNext** の説明が顧客に届きやすい

[^1]: 例: namuamidabu.com（4ヶ月体験レビュー）/ taiwanlingo.com（3ヶ月レビュー）/ ltl-school.jp

## A-2. HelloChinese

### 概要

| 項目 | 内容 |
|---|---|
| 運営 | HelloChinese Team（中国系） |
| 対象レベル | 完全初心者〜中級者 |
| 料金 | 無料 / 有料版 年額約¥8,000 |
| 特徴 | ピンインから学べる、実写動画リスニング、AI発音判定 |

### 強み

- **発音とピンインの基礎が手厚い**
- 実写動画（ネイティブが話す表情・口の動き）
- 手書き練習
- 詳しい日本語の文法解説
- HSK 1〜5級対応

### 弱み

- カリキュラム固定（個人化なし）
- 「自分の現場で必要なフレーズ」は出てこない
- スピーキング練習はAI発音判定どまり

### 当方との位置関係

- **基礎を支援する間接競合**
- 「HelloChineseで基礎を作り、SayItNextで現場に持ち込む」**併用ペアリング**が成立する可能性
- 直接奪い合うのは「中級〜上級ユーザーの予算と時間」

## A-3. SuperChinese

### 概要

| 項目 | 内容 |
|---|---|
| 運営 | SuperChinese Team（中国系） |
| 対象レベル | 初心者〜上級者 |
| 料金 | 無料 / 有料版 年額約¥12,000 |
| 特徴 | AIチューター "CHAO" による1対1会話、画像認識、スピーキングチャレンジ |

### 強み

- AIによる**パーソナライズ学習**（個々のレベルに最適化）
- HSK 1〜5級の体系的カリキュラム
- スピーキング機能の充実
- 画像認識など先端機能

### 弱み

- パーソナライズは「**レベル**」のみで、「**現場・文脈**」には対応していない
- 「自分が今日言いたかった一言」を入力する機能はない
- 料金が高め（年¥12,000）

### 当方との位置関係

- **最も近い競合**の一つ
- 「AIパーソナライズ」を打ち出している点で、メッセージが衝突しやすい
- ただし、**パーソナライズの方向性が違う**（カリキュラム vs フレーズ単位）
- 差別化メッセージ: 「**レベルではなく、あなたの現場に最適化**」

# B. AI翻訳・調べる手段

## B-1. ChatGPT / Gemini

### 概要

中国語学習における**最大の代替手段**。実質的に**最大の競合**である可能性が高い。

### 顧客の使用パターン

1. 言いたかったことを質問して翻訳してもらう
2. 文法を確認する
3. ロールプレイ会話練習
4. 例文を生成してもらう
5. 試験対策の問題を出してもらう

### 強み

- 24時間対応、即時
- 無料 or 月額¥2,400程度
- 圧倒的な汎用性
- 自然な会話形式

### 弱み（重要）

| # | 弱点 | 我々の対応 |
|---|---|---|
| 1 | **ハルシネーション**（もっともらしい嘘） | 専用プロンプトで品質保証 |
| 2 | **定着しない**（調べて終わり） | SRS で反復ドリル化 |
| 3 | **保存・整理されない** | 自分用フレーズ集に自動保存 |
| 4 | 学習体験のフローがない | フレーズ追加→ドリル→復習の動線 |
| 5 | 中国特有の文化・古典に弱い | （SayItNextも同じ制約） |

### 当方との位置関係

- **最大の競合** — 顧客の「最初に試す手段」がChatGPT
- 差別化メッセージ: 「**ChatGPTで足りるのは『調べる』まで。次に同じ場面で口から出るかは別問題です**」
- ChatGPTを否定するのではなく、「**ChatGPTの次の一歩**」として位置づけるのが戦略上有利
- 既に確定したコピー(`customer-discovery.md`)と整合

---

## B-2. DeepL / Google翻訳

### 概要

純粋な翻訳ツール。学習効果ゼロ。

### 当方との位置関係

- 直接競合ではない
- ただし、**「翻訳できるならいいや」**で学習を始めない人を生んでいる
- 我々は「翻訳の次」を提供 → 認知の上では翻訳ツールとは差別化が容易

# D. SRS・フラッシュカード系

## D-1. Anki

### 概要

| 項目 | 内容 |
|---|---|
| 運営 | オープンソース |
| 中国語デッキ | 多数（HSK 1-6, 5000語、11400語など） |
| 料金 | デスクトップ無料 / iOS¥3,000程度（買い切り） |
| 特徴 | SM-2ベースSRS、無制限カスタマイズ |

### 人気の中国語デッキ

- **Hanping Chinese HSK (1-6)** — 5000枚、評価58件
- **HSK 1-6 official vocabulary** — 4997枚
- **Mega Mandarin HSK 3.0** — 11400+語、音声付き
- **Chinese HSK 5000 Frequency Order** — 頻度順

### 強み

- **SRSの科学的根拠**が確立されている
- 完全カスタマイズ可能
- オフライン動作

### 弱み

- **設計の敷居が高い**（カード作成、デッキ管理）
- **AI生成がない**（フレーズは自分で書くか既存デッキを使う）
- **音声・解説は手動**
- **既存デッキは「教科書」中心**（現場フレーズではない）

### 当方との位置関係

- **SRS技術の元祖** — 我々もSM-2ベースを採用
- 差別化: 「**Ankiの設計時間ゼロ + AI生成 + 現場最適化**」
- 顧客タイプ: Anki中級者は**アーリーアダプター候補**（SRSの価値を既に知っている）

# ポジショニング図

`customer-discovery.md` の **5-3** と同一の軸定義に統一した。

```
                  反射定着できる
                       │
                       │
  Anki, SuperChinese ──┼── SayItNext (●)
                       │
                       │
  教材中心 ────────────┼──────────── ユーザー固有
                       │
                       │
  Duolingo, HelloCN ──┼── ChatGPT, DeepL
                       │
                       │
                  単発理解で終わる
```

- **右上（ユーザー固有 × 反射定着できる）**が空白に見える → SayItNext の狙い。
- **左下**: 教材が決まっており、利用のたびに「理解」はしても**定着フローは限定的**（Duolingo / HelloChinese 等）。
- **左上**: 教材（または公式系カリキュラム）寄りだが、SRS 等で**反復定着**は強い（Anki / SuperChinese 等）。
- **右下**: 質問は**ユーザー固有**だが、その場の回答で終わり、**SRS まで組み込まれない**（ChatGPT / DeepL）。

→ 既存リーダーは「教材中心ビジネス」と「単発利用」のどちらかに寄りやすく、**ユーザー固有と反射定着を同時に製品化する**ところに構造的な隙が出やすい、という読み。

# ブルーオーシャン（未開拓の機会）

## 機会1: 「言いたかった瞬間」起点の設計

ほぼ全競合が「**教えたいことを順番に教える**」設計。  
我々は「**ユーザーが言いたかったことから始まる**」設計。

→ 顧客の**痛みのある瞬間**が起点なので、モチベーション維持が構造的に強い。

## 機会2: AI生成 × SRSの組み合わせ

ChatGPTはAI生成があるが定着しない。AnkiはSRSがあるがカード作成が重い。  
**この2つを繋ぐ**サービスは現時点で見つからない。

## 機会3: 現場×コンテキストの最適化

「ポーカー」「中国出張」「マカオ生活」など**特定の文脈**に最適化したサービスは皆無。  
ニッチで深い顧客痛みを直接捉えられる。

## 機会4: 日本語UI × 中国語特化 × 個人化

中国系アプリ（HelloChinese, SuperChinese）は中国語特化だが、**日本人特有の課題感**には対応しきれない。  
日本人作・日本語UI・日本人視点のアプリは**れんペルソナに刺さりやすい**。

# 脅威（要警戒）

## 脅威1: ChatGPTの進化（最大の脅威）

GPT-5 / Gemini 3.0 が「**生成 + 記憶 + リマインド**」を一気に内包し始めると、独立アプリの優位性が削られる可能性。

### 対応策

- 「**現場×日本人×中国語**」という3点絞り込みで差別化を死守
- AIモデル自体ではなく「**学習体験のフロー**」が資産になる設計
- データ（ユーザー固有フレーズ、現場知見）を資産化

## 脅威2: Duolingoの中級者向け強化

Duolingoが「Duolingo Max」などで個人化を強化する可能性。

### 対応策

- 中級〜上級者の「**自分の現場で困っている人**」セグメントで深く根を張る
- 認知度競争で勝つのは難しいので、**コミュニティ・伝道力**で勝つ

## 脅威3: SuperChineseのAIパーソナライズ深化

CHAOがフレーズ単位のパーソナライズに進化する可能性。

### 対応策
- 「**現場・文脈の最適化**」で差別化を死守
- ポーカー・出張など**特定ドメイン**のフレーズデータを資産化

## 脅威4: 中国系AI学習アプリの日本進出

DeepSeekやBaichuan系AIを使った中国製アプリが日本市場に進出する可能性。

### 対応策

- 日本人視点の「**ナメられたくない**」「**評価されたい**」の感情軸で勝負
- 中国系アプリには絶対作れないメッセージング

# 我々の差別化メッセージ（既存と整合）

| 場面 | コピー |
|---|---|
| 短文（X用） | ChatGPTで調べた100フレーズ、何個覚えてますか？ |
| 中文（LP用） | ChatGPTで足りるのは「調べる」まで。次に同じ場面で口から出るかは別問題です。 |
| 長文（note用） | なぜChatGPTで中国語を調べても、現場で言えるようにならないのか。それは「調べる」と「覚える」「使う」の間に、誰も橋をかけていないからです。 |
| 訴求軸 | **今日言えなかったことを、明日言えるようになる。** |

→ 競合調査結果と完全に整合。**差別化メッセージは強い**。

# Phase 2 で確認すべき検証ポイント

## 検証1: 顧客は今、何を使っているか

5〜10件のインタビューで、**実際の利用率分布**を把握する。

仮説:
- 70%以上が ChatGPT または翻訳アプリを併用
- 50%以上が Duolingo の挫折経験あり
- 20%以下が SuperChinese / HelloChinese を継続中
- 5%以下が Anki を使っている

## 検証2: 「Duolingoで続かなかった」エピソードの実在

ほぼ全員が持っているか、それともサンプルの一部か。

## 検証3: ChatGPTの「定着しない」痛みの深さ

「便利だけど忘れる」を**痛み**として感じているか、**まあそんなもの**と諦めているか。

## 検証4: 「自分用フレーズ集」コンセプトの理解

ペルソナに伝えた時、「**それ欲しい**」と即理解されるか、**説明が必要**か。

# 競合分析からの戦略的示唆

## 短期（〜1ヶ月）

1. **訴求の主軸**は「ChatGPT との対比」に置く（最も強力な比較対象）
2. **Duolingo 挫折経験者**を主要ターゲットにメッセージ設計
3. **アーリーアダプター候補**は「ChatGPT × Anki」併用ユーザー
4. インタビューで「実際に使っているアプリ」を必ず聞く

## 中期（〜6ヶ月）

1. **ポジショニング**: 「ユーザー固有 × 反射定着できる」の右上を死守
2. 「ポーカー」「中国出張」など**特定文脈フレーズデータ**を資産化
3. SuperChinese 等との**差別化**を明確にしたコンテンツ（note記事など）

## 長期（〜2年）

1. 多言語拡張時、Duolingoとの**真っ向勝負**は避け、「現場特化」を維持
2. 競合がAI生成×SRSを追ってきた場合に備え、**コミュニティ・ユーザーデータ**で参入障壁を作る
3. 「ChatGPT進化」シナリオに備え、AIモデル非依存の**学習体験の独自性**を強化
